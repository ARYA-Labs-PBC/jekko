Jekko: a Rust-native agentic coding assistant.

Run `jekko` with no arguments to launch the TUI. Pass a
prompt with `jekko run "..."` for one-shot execution,
or `jekko serve` to start a headless HTTP server.

Examples:
jekko                       # launch the TUI
jekko run "refactor foo"  # one-shot run
jekko serve --port 8080     # headless server
jekko session list          # list sessions
jekko keys set OPENAI       # set an API key

Usage: jekko [OPTIONS] [PATH] [COMMAND]

Commands:
  tui        Launch the interactive TUI explicitly. Same as running `jekko` with no arguments
  run        One-shot non-interactive run
  serve      Start the HTTP server
  session    Manage sessions
  providers  Manage AI providers and credentials
  models     List or inspect available models
  keys       Manage canonical model API keys
  agent      Manage agents
  mcp        Manage MCP (Model Context Protocol) servers
  acp        ACP (Agent Client Protocol) integration
  jankurai   Jankurai integration commands
  daemon     Manage the background daemon
  plugin     Manage plugins
  debug      Diagnostic and debug utilities
  import     Import sessions from a backup
  export     Export sessions to a file
  stats      Usage statistics
  pr         Pull-request helpers (passthrough to xtask)
  github     GitHub event helpers (passthrough to xtask)
  db         Database tools
  upgrade    Upgrade Jekko in place
  uninstall  Uninstall Jekko and remove data directories

Arguments:
  [PATH]
          Optional working directory; defaults to the current directory

Options:
      --print-logs
          Print structured logs to stderr

      --log-level <LEVEL>
          Override the log level (TRACE, DEBUG, INFO, WARN, ERROR)

      --pure
          Disable external plugins

      --headless
          Run without entering raw mode / alt-screen (PTY tests, headless smoke)

      --cwd <PATH>
          Override the working directory before any I/O is performed

  -c, --continue
          Resume the last session

  -s <SESSION_ID>
          Open a specific session by id

  -h, --help
          Print help (see a summary with '-h')

  -V, --version
          Print version
