# 2. Why Ratatui is the right fit

Ratatui’s immediate-mode rendering model is better aligned with a high-performance coding cockpit than the current JSX-style terminal UI. Ratatui expects the app to own the event loop, state model, and redraw cycle, while Ratatui handles drawing widgets into terminal frames; this gives you precise control over layout, input, animation, terminal resize behavior, and frame composition. ([Ratatui][2])

Ratatui also has built-in support for rich widgets and layouts that map directly to the desired app:

```txt
Left Intelligence Rail       -> List, Table, Gauge, Sparkline
Conversation Canvas          -> Paragraph, Block, Scrollbar, custom widgets
Command Deck                 -> ratatui-textarea + custom footer chips
Jnoccio/Jankurai dashboards   -> Chart, Gauge, Sparkline, Table
Diff viewer                  -> custom line renderer + styled spans
Theme picker                 -> StatefulList / Table
Permission cards             -> centered popup Block + selectable buttons
```

The official Ratatui examples and docs show style/color examples, RGB color support, layout examples, widget examples, and app demos. Ratatui 0.30.0 is listed as the current docs.rs crate version, and Ratatui 0.30+ also supports multiple Crossterm major versions through feature flags, which matters for dependency compatibility. ([Docs.rs][3])

## `src/app.rs`

```rust
use anyhow::Result;
use crossterm::event::{self, Event as CrosstermEvent, KeyCode, KeyEventKind};
use ratatui::{DefaultTerminal, Frame};
use std::time::{Duration, Instant};

use crate::{
    backend::client::BackendClient,
    event::{AppEvent, EventBus},
    keymap::{Action, Keymap},
    state::AppState,
    ui,
};

pub struct App {
    state: AppState,
    client: BackendClient,
    events: EventBus,
    keymap: Keymap,
    last_tick: Instant,
}

impl App {
    pub async fn boot() -> Result<Self> {
        let config = crate::config::Config::load()?;
        let client = BackendClient::new(config.backend_url.clone());

        let state = AppState::new(config);
        let events = EventBus::spawn(client.clone()).await?;

        Ok(Self {
            state,
            client,
            events,
            keymap: Keymap::default(),
            last_tick: Instant::now(),
        })
    }

    pub async fn run(&mut self, terminal: &mut DefaultTerminal) -> Result<()> {
        loop {
            terminal.draw(|frame| self.render(frame))?;

            if self.state.should_quit {
                break;
            }

            self.pump_terminal_events().await?;
            self.pump_backend_events().await?;
            self.tick().await?;
        }

        Ok(())
    }

    fn render(&mut self, frame: &mut Frame) {
        ui::render(frame, &mut self.state);
    }

    async fn pump_terminal_events(&mut self) -> Result<()> {
        if !event::poll(Duration::from_millis(8))? {
            return Ok(());
        }

        let CrosstermEvent::Key(key) = event::read()? else {
            return Ok(());
        };

        if key.kind != KeyEventKind::Press {
            return Ok(());
        }

        let action = self.keymap.resolve(key, &self.state);

        match action {
            Action::Quit => self.state.should_quit = true,
            Action::ToggleTheme => self.state.theme.toggle_mode(),
            Action::ToggleLeftRail => self.state.layout.left_rail_open = !self.state.layout.left_rail_open,
            Action::OpenCommandPalette => self.state.dialog.open_command_palette(),
            Action::SubmitPrompt => {
                let prompt = self.state.command_deck.take_text();
                self.client.send_prompt(self.state.current_session_id(), prompt).await?;
            }
            Action::Input(key) => self.state.command_deck.input(key),
            Action::Noop => {}
        }

        Ok(())
    }

    async fn pump_backend_events(&mut self) -> Result<()> {
        while let Some(event) = self.events.try_recv() {
            self.state.apply_backend_event(event);
        }

        Ok(())
    }

    async fn tick(&mut self) -> Result<()> {
        let now = Instant::now();

        if now.duration_since(self.last_tick) >= Duration::from_millis(100) {
            self.state.animations.tick(now);
            self.last_tick = now;
        }

        Ok(())
    }
}
```

This is the big benefit of Ratatui: the entire UI becomes a deterministic function of `AppState`.

```rust
    pub body: Rect,
    pub left_rail: Rect,

    pub status_bar: Rect,

pub fn compute(area: Rect, left_rail_open: bool, left_rail_width: u16) -> AppLayout {

        .direction(Direction::Vertical)
        .constraints([
            Constraint::Length(1), // top HUD
            Constraint::Min(8),    // main body
            Constraint::Length(6), // command deck
            Constraint::Length(1), // status bar
        ])
        .split(area);

    let body_chunks = if left_rail_open {
        Layout::default()
            .direction(Direction::Horizontal)
            .constraints([
                Constraint::Length(left_rail_width),
                Constraint::Min(40),
            ])
            .split(vertical[1])
    } else {
        Layout::default()
            .direction(Direction::Horizontal)
            .constraints([
                Constraint::Length(0),
                Constraint::Min(40),
            ])
            .split(vertical[1])
    };

    AppLayout {
        top_hud: vertical[0],
        body: vertical[1],
        left_rail: body_chunks[0],
        conversation: body_chunks[1],
        command_deck: vertical[2],
        status_bar: vertical[3],
```

# 9. Keyboard shortcuts

## `src/keymap.rs`

```rust
use crossterm::event::{KeyCode, KeyEvent, KeyModifiers};

#[derive(Debug, Clone)]
pub enum Action {
    Quit,
    ToggleTheme,
    ToggleLeftRail,
    OpenCommandPalette,
    SubmitPrompt,
    Input(KeyEvent),
    Noop,
}

#[derive(Debug, Clone)]
pub struct Keymap {
    pub theme_toggle: (KeyModifiers, KeyCode),
    pub command_palette: (KeyModifiers, KeyCode),
    pub left_rail_toggle: (KeyModifiers, KeyCode),
}

impl Default for Keymap {
    fn default() -> Self {
        Self {
            theme_toggle: (KeyModifiers::CONTROL, KeyCode::Char('t')),
            command_palette: (KeyModifiers::CONTROL, KeyCode::Char('/')),
            left_rail_toggle: (KeyModifiers::CONTROL, KeyCode::Char('b')),
        }
    }
}

impl Keymap {
    pub fn resolve(&self, key: KeyEvent, _state: &crate::state::AppState) -> Action {
        match (key.modifiers, key.code) {
            (KeyModifiers::CONTROL, KeyCode::Char('c')) => Action::Quit,
            (m, c) if (m, c) == self.theme_toggle => Action::ToggleTheme,
            (m, c) if (m, c) == self.command_palette => Action::OpenCommandPalette,
            (m, c) if (m, c) == self.left_rail_toggle => Action::ToggleLeftRail,
            (_, KeyCode::Enter) => Action::SubmitPrompt,
            _ => Action::Input(key),
        }
    }
}
```

```rust
) {
    let theme = &state.theme;
    let Some(message) = state.message_by_id(&message_id) else {
        return;
    };

    let tone = match message.role {
        MessageRole::User => theme.primary,
        MessageRole::Assistant => theme.secondary,
        MessageRole::System => theme.muted,
        MessageRole::Error => theme.error,
    };

    let block = Block::default()
        .borders(Borders::ALL)
        .border_style(ratatui::style::Style::default().fg(theme.border_subtle))
        .style(theme.card())
        .title(Line::from(vec![
            Span::styled(message.role.label(), ratatui::style::Style::default().fg(tone)),
            Span::styled(" · ", theme.muted()),
            Span::styled(message.timestamp_label(), theme.muted()),
        ]));

    let inner = block.inner(area);
    frame.render_widget(block, area);

    let text = Text::from(message.render_lines(theme));

    let paragraph = Paragraph::new(text)
        .wrap(Wrap { trim: false })
        .style(ratatui::style::Style::default().fg(theme.text).bg(theme.panel));

    frame.render_widget(paragraph, inner);
}
```

This replaces the current OpenCode-like left-stripe visual identity with full card-based Ratatui rendering.

# 15. Dialogs and command palette

## `src/ui/dialogs.rs`

```rust
use ratatui::{
    layout::{Alignment, Constraint, Direction, Layout, Rect},
    text::{Line, Span},
    widgets::{Block, Borders, Clear, List, ListItem, Paragraph},
    Frame,
};

use crate::state::{AppState, DialogKind};

pub fn render(frame: &mut Frame, state: &mut AppState) {
    let Some(dialog) = state.dialog.current() else {
        return;
    };

    let area = centered_rect(70, 70, frame.area());
    let theme = &state.theme;

    frame.render_widget(Clear, area);

    let block = Block::default()
        .borders(Borders::ALL)
        .border_style(ratatui::style::Style::default().fg(theme.border_active))
        .style(theme.card())
        .title(Span::styled(dialog.title(), theme.title()));

    let inner = block.inner(area);
    frame.render_widget(block, area);

    match dialog.kind() {
        DialogKind::CommandPalette => render_command_palette(frame, inner, state),
        DialogKind::ThemePicker => render_theme_picker(frame, inner, state),
        DialogKind::Permission => render_permission(frame, inner, state),
        DialogKind::Question => render_question(frame, inner, state),
    }
}

fn render_command_palette(frame: &mut Frame, area: Rect, state: &AppState) {
    let theme = &state.theme;

    let items = state
        .commands
        .filtered()
        .iter()
        .map(|cmd| {
            ListItem::new(Line::from(vec![
                Span::styled(&cmd.title, theme.title()),
                Span::styled("  ", theme.muted()),
                Span::styled(cmd.shortcut_label(), theme.muted()),
            ]))
        })
        .collect::<Vec<_>>();

    let list = List::new(items)
        .highlight_style(theme.selected())
        .highlight_symbol("◆ ");

    frame.render_stateful_widget(list, area, &mut state.commands.list_state.clone());
}

fn centered_rect(percent_x: u16, percent_y: u16, area: Rect) -> Rect {
    let vertical = Layout::default()
        .direction(Direction::Vertical)
        .constraints([
            Constraint::Percentage((100 - percent_y) / 2),
            Constraint::Percentage(percent_y),
            Constraint::Percentage((100 - percent_y) / 2),
        ])
        .split(area);

    Layout::default()
        .direction(Direction::Horizontal)
        .constraints([
            Constraint::Percentage((100 - percent_x) / 2),
            Constraint::Percentage(percent_x),
            Constraint::Percentage((100 - percent_x) / 2),
        ])
        .split(vertical[1])[1]
}
```

This gives you command palette, model picker, theme picker, permission prompt, and question prompt as one consistent premium modal system.

# 16. Backend client

## `src/backend/client.rs`

```rust
use anyhow::Result;
use reqwest::Client;
use serde::{Deserialize, Serialize};

#[derive(Clone)]
pub struct BackendClient {
    base_url: String,
    http: Client,
}

impl BackendClient {
    pub fn new(base_url: String) -> Self {
        Self {
            base_url,
            http: Client::new(),
        }
    }

    pub async fn list_sessions(&self) -> Result<Vec<Session>> {
        let url = format!("{}/session", self.base_url);

        Ok(self
            .http
            .get(url)
            .send()
            .await?
            .error_for_status()?
            .json()
            .await?)
    }

    pub async fn send_prompt(&self, session_id: &str, text: String) -> Result<()> {
        let url = format!("{}/session/{}/prompt", self.base_url, session_id);

        self.http
            .post(url)
            .json(&PromptRequest { text })
            .send()
            .await?
            .error_for_status()?;

        Ok(())
    }

    pub async fn abort_session(&self, session_id: &str) -> Result<()> {
        let url = format!("{}/session/{}/abort", self.base_url, session_id);

        self.http
            .post(url)
            .send()
            .await?
            .error_for_status()?;

        Ok(())
    }
}

#[derive(Debug, Clone, Deserialize)]
pub struct Session {
    pub id: String,
    pub title: String,
}

#[derive(Debug, Serialize)]
struct PromptRequest {
    text: String,
}
```

```rust
impl AppState {
    pub fn apply_backend_event(&mut self, event: BackendEvent) {
        match event {
            BackendEvent::SessionUpdated(session) => self.sessions.upsert(session),
            BackendEvent::MessagePartUpdated(part) => self.messages.upsert_part(part),
            BackendEvent::PermissionCreated(permission) => self.permissions.push(permission),
```
