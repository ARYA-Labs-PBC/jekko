# Jekko OpenTUI/Bun/Vite Removal and Ratatui Migration Engineering Plan

## 1. Executive summary

The uploaded project snapshot is heavily centered on a TypeScript/OpenTUI/Solid TUI stack, with Bun used as package manager, runtime, build tool, test runner, script runner, and plugin dependency installer. The right migration is **not** to shim OpenTUI or gradually wrap TSX components. The clean architecture is a **Rust-first CLI/TUI** built with **Ratatui + Crossterm + Tokio**, with Jekko’s domain data, SDK sync, session rendering, command system, plugin slots, prompts, dialogs, and dashboards ported into Rust modules.

I found **no Vite references** in the uploaded snapshot. Still, the migration should add hard CI guards to prevent Vite from re-entering the repo.

Ratatui is a lightweight Rust TUI crate with widgets/utilities for complex terminal UIs; it expects the app to own state, redraw frames, and use a backend such as Crossterm for input and terminal mode management. That fits this migration well because the current OpenTUI/Solid app already has centralized route, state, keybind, dialog, sync, and plugin concepts that can be converted into an Elm/component-style Rust architecture. Ratatui’s own docs describe component architecture as state + event handling + update + render colocated per component, and its event-handling docs make clear that Ratatui does not capture events itself; the application/backend does that work. ([Docs.rs][1])

The hardest areas are:

1. **Prompt replacement**: current prompt depends on OpenTUI `TextareaRenderable`, extmarks, paste decoding, virtual prompt parts, autocomplete overlays, and `opentui-spinner`.
2. **Plugin UI replacement**: current external TUI plugins are TSX/Solid/OpenTUI modules. A 100% OpenTUI/Bun removal means these plugin UIs cannot remain as-is.
3. **Build/CI/Nix/tooling removal**: Bun is present across `package.json`, workflows, `ops/ci`, Nix, scripts, tests, and local plugin install behavior.
4. **Session transcript fidelity**: assistant/user message renderers, tool renderers, diff blocks, scroll behavior, permissions, questions, footer bands, and responsive sidebars need careful Ratatui equivalents.

The target should be:

```text
Cargo workspace
  crates/jekko-cli       # binary entrypoint, CLI args, command dispatch
  crates/jekko-tui       # Ratatui UI, event loop, screens, widgets
  crates/jekko-core      # domain models, config, session data, theme parsing
  crates/jekko-sdk       # HTTP/WebSocket client and event sync
  crates/jekko-plugin-api # Rust/declarative plugin protocol
  crates/xtask           # repo automation replacing Bun scripts
```

If the repo still needs JavaScript packages for SDK publishing, keep those packages, but they must use **Node/npm/pnpm/yarn**, not Bun, and must not be used for the TUI.

## 3. Migration principles

### 3.1 Do not port TSX one-to-one mechanically

OpenTUI/Solid is declarative and reactive. Ratatui is immediate-mode rendering. The right model is:

```text
AppState
  + Event
  -> Action
  -> State update
  -> Draw full frame
```

Use components for boundaries, but keep the frame render deterministic. Ratatui’s official docs describe a component approach where each component owns state, events, updates, and rendering. ([Ratatui][2])

### 3.2 Use Crossterm as the terminal backend

The current app uses alternate screen, raw keyboard mode, mouse capture, terminal title changes, Ctrl+C handling, and startup/fatal restoration. Crossterm is the canonical backend choice here. Ratatui’s Crossterm backend docs show that apps usually enable raw mode, enter alternate screen, draw with `Terminal`, and restore the terminal on exit. ([Docs.rs][3])

### 3.3 Keep pixel-perfectness through tokens, not CSS-like shims

OpenTUI supports layout semantics like `box`, `flexDirection`, `gap`, `padding`, `border`, `zIndex`, `backgroundColor`, `scrollbox`, and `textarea`. Ratatui does not have CSS/flexbox by default. Recreate the visible UI using:

```text
ratatui::layout::{Layout, Constraint, Direction, Rect}
ratatui::widgets::{Block, Paragraph, List, Clear, Gauge, Scrollbar}
ratatui::text::{Line, Span, Text}
ratatui::style::{Style, Color, Modifier}
unicode-width for cell-accurate truncation
custom helper functions for padding, centering, overlays, and clipped rects
```

The result can be very close visually, but the implementation should be idiomatic Ratatui, not a homegrown OpenTUI clone.

### 3.4 Split “data services” from “view components”

Current Solid providers should become ordinary Rust state/services:

```text
ArgsProvider              -> CliArgs / AppConfig
ExitProvider              -> ExitState / AppAction::Quit
KVProvider                -> KvStore
ToastProvider             -> ToastState
RouteProvider             -> Router
TuiConfigProvider         -> TuiConfig
SDKProvider               -> SdkClient
ProjectProvider           -> ProjectState
SyncProvider / SyncV2     -> SyncStore + background tasks
ThemeProvider             -> ThemeStore
LocalProvider             -> LocalSelectionState
KeybindProvider           -> Keymap
PromptStashProvider       -> PromptStash
DialogProvider            -> DialogStack
CommandProvider           -> CommandRegistry
FrecencyProvider          -> FrecencyStore
PromptHistoryProvider     -> PromptHistory
EditorContextProvider     -> EditorContext
```

### 5.3 `packages/plugin`

Current `packages/plugin/src/tui.ts` exports OpenTUI/Solid plugin API types. That cannot remain under a 100% OpenTUI removal.

Replace it with one of two approaches:

#### Recommended: declarative plugin UI protocol

Keep external plugins, but do not allow TSX rendering. A plugin may register commands, routes, slots, and declarative view models:

```ts
// future JS plugin API, if JS plugins are retained without Bun
export type TuiView =
  | { type: "text"; text: string; style?: TuiStyle }
  | { type: "row"; children: TuiView[] }
  | { type: "column"; children: TuiView[] }
  | { type: "panel"; title?: string; children: TuiView[] }
  | { type: "list"; rows: TuiListRow[] }

export type TuiPlugin = {
  id: string
  commands?: TuiCommand[]
  routes?: TuiRoute[]
  slots?: TuiSlotContribution[]
  themes?: string[]
}
```

Rust host renders these view models with Ratatui. This preserves plugin concepts while removing OpenTUI/Solid.

#### Alternative: Rust-only plugin API

Use compiled internal plugins and a future dynamic/plugin ABI only if truly needed. For near-term safety, make all existing built-in feature plugins first-class Rust modules.

## 7. Ratatui architecture and core code

### 7.1 Terminal lifecycle

```rust
// crates/jekko-tui/src/terminal.rs

use std::io::{self, Stdout};

use anyhow::Result;
use crossterm::{
    cursor::{Hide, Show},
    event::{DisableMouseCapture, EnableMouseCapture},
    execute,
    terminal::{
        disable_raw_mode, enable_raw_mode, EnterAlternateScreen, LeaveAlternateScreen,
    },
};
use ratatui::{backend::CrosstermBackend, Terminal};

pub type TuiTerminal = Terminal<CrosstermBackend<Stdout>>;

pub struct TerminalGuard {
    terminal: TuiTerminal,
}

impl TerminalGuard {
    pub fn enter(enable_mouse: bool) -> Result<Self> {
        enable_raw_mode()?;

        let mut stdout = io::stdout();
        execute!(stdout, EnterAlternateScreen, Hide)?;

        if enable_mouse {
            execute!(io::stdout(), EnableMouseCapture)?;
        }

        let backend = CrosstermBackend::new(io::stdout());
        let mut terminal = Terminal::new(backend)?;
        terminal.clear()?;

        Ok(Self { terminal })
    }

    pub fn terminal_mut(&mut self) -> &mut TuiTerminal {
        &mut self.terminal
    }
}

impl Drop for TerminalGuard {
    fn drop(&mut self) {
        let _ = disable_raw_mode();
        let _ = execute!(
            io::stdout(),
            DisableMouseCapture,
            Show,
            LeaveAlternateScreen
        );
    }
}
```

### 7.2 App event loop

```rust
// crates/jekko-tui/src/app.rs

use std::time::{Duration, Instant};

use anyhow::Result;
use crossterm::event::{self, Event as CrosstermEvent, KeyEventKind};
use tokio::sync::mpsc;

use crate::{
    action::Action,
    event::AppEvent,
    state::AppState,
    terminal::TerminalGuard,
};

const FRAME_INTERVAL: Duration = Duration::from_millis(16); // close to current 60fps target

pub async fn run(mut state: AppState) -> Result<()> {
    let mut terminal = TerminalGuard::enter(state.config.mouse)?;
    let (tx, mut rx) = mpsc::unbounded_channel::<AppEvent>();

    spawn_input_task(tx.clone());
    spawn_tick_task(tx.clone());
    spawn_sync_task(tx.clone(), state.sdk.clone());

    let mut last_draw = Instant::now();

    loop {
        while let Ok(event) = rx.try_recv() {
            let actions = state.handle_event(event);
            for action in actions {
                state.update(action)?;
            }
        }

        if last_draw.elapsed() >= FRAME_INTERVAL || state.needs_redraw {
            terminal.terminal_mut().draw(|frame| {
                state.render(frame);
            })?;
            state.needs_redraw = false;
            last_draw = Instant::now();
        }

        if state.should_quit {
            break;
        }

        if let Some(event) = rx.recv().await {
            let actions = state.handle_event(event);
            for action in actions {
                state.update(action)?;
            }
        }
    }

    Ok(())
}

fn spawn_input_task(tx: mpsc::UnboundedSender<AppEvent>) {
    std::thread::spawn(move || loop {
        match event::read() {
            Ok(CrosstermEvent::Key(key)) if key.kind == KeyEventKind::Press => {
                let _ = tx.send(AppEvent::Key(key));
            }
            Ok(CrosstermEvent::Mouse(mouse)) => {
                let _ = tx.send(AppEvent::Mouse(mouse));
            }
            Ok(CrosstermEvent::Resize(width, height)) => {
                let _ = tx.send(AppEvent::Resize { width, height });
            }
            Ok(_) => {}
            Err(error) => {
                let _ = tx.send(AppEvent::Error(error.to_string()));
                break;
            }
        }
    });
}

fn spawn_tick_task(tx: mpsc::UnboundedSender<AppEvent>) {
    tokio::spawn(async move {
        let mut interval = tokio::time::interval(Duration::from_millis(90));
        loop {
            interval.tick().await;
            if tx.send(AppEvent::Tick).is_err() {
                break;
            }
        }
    });
}

fn spawn_sync_task(tx: mpsc::UnboundedSender<AppEvent>, sdk: crate::services::SdkHandle) {
    tokio::spawn(async move {
        sdk.run_event_stream(tx).await;
    });
}
```

### 8.2 Ratatui color model

```rust
// crates/jekko-core/src/theme/color.rs

use serde::{Deserialize, Serialize};

#[derive(Debug, Clone, Copy, PartialEq, Eq, Serialize, Deserialize)]
pub struct Rgba {
    pub r: u8,
    pub g: u8,
    pub b: u8,
    pub a: u8,
}

impl Rgba {
    pub fn from_hex(input: &str) -> anyhow::Result<Self> {
        let hex = input.trim_start_matches('#');
        let parse = |range: std::ops::Range<usize>| {
            u8::from_str_radix(&hex[range], 16)
        };

        let (r, g, b, a) = match hex.len() {
            6 => (parse(0..2)?, parse(2..4)?, parse(4..6)?, 255),
            8 => (parse(0..2)?, parse(2..4)?, parse(4..6)?, parse(6..8)?),
            _ => anyhow::bail!("invalid hex color: {input}"),
        };

        Ok(Self { r, g, b, a })
    }

    pub fn to_ratatui(self) -> ratatui::style::Color {
        ratatui::style::Color::Rgb(self.r, self.g, self.b)
    }

    pub fn blend_over(self, bg: Rgba) -> Rgba {
        let alpha = self.a as f32 / 255.0;
        let inv = 1.0 - alpha;
        Rgba {
            r: ((self.r as f32 * alpha) + (bg.r as f32 * inv)).round() as u8,
            g: ((self.g as f32 * alpha) + (bg.g as f32 * inv)).round() as u8,
            b: ((self.b as f32 * alpha) + (bg.b as f32 * inv)).round() as u8,
            a: 255,
        }
    }
}
```

Important: terminals do not support true alpha blending per cell. Any OpenTUI transparent/alpha colors must be pre-blended against the active background.

```rust
// crates/jekko-tui/src/screens/splash.rs

use std::time::{Duration, Instant};

use ratatui::{
    Frame,
    layout::{Alignment, Constraint, Direction, Layout, Rect},
    style::{Modifier, Style},
    text::{Line, Span},
    widgets::{Block, Paragraph},
};

use crate::{state::AppState, ui::rect};

const MIN_SHOW: Duration = Duration::from_millis(2400);
const HARD_CAP: Duration = Duration::from_millis(15_000);
const PULSE_PERIOD_MS: u128 = 4600;
const SPINNER: [&str; 10] = ["⠋", "⠙", "⠹", "⠸", "⠼", "⠴", "⠦", "⠧", "⠇", "⠏"];
const NEVER_HUMAN: &str = "N · E · V · E · R · H · U · M · A · N";

#[derive(Debug, Clone)]
pub struct SplashState {
    pub mounted_at: Instant,
    pub revealed: usize,
    pub dismissed: bool,
}

impl Default for SplashState {
    fn default() -> Self {
        Self {
            mounted_at: Instant::now(),
            revealed: 1,
            dismissed: false,
        }
    }
}

#[derive(Debug, Clone)]
pub enum BootStatus {
    Running,
    Done,
    Warn,
    Error,
}

#[derive(Debug, Clone)]
pub struct BootLine {
    pub phase: &'static str,
    pub message: String,
    pub status: BootStatus,
}

pub fn render(frame: &mut Frame, area: Rect, splash: &SplashState, state: &AppState) {
    let theme = &state.theme;

    frame.render_widget(
        Block::default().style(Style::default().bg(theme.background.to_ratatui())),
        area,
    );

    let chunks = Layout::default()
        .direction(Direction::Horizontal)
        .constraints([Constraint::Percentage(60), Constraint::Percentage(40)])
        .split(area);

    render_boot_log(frame, chunks[0], splash, state);
    render_brand(frame, chunks[1], splash, state);
}

fn render_boot_log(frame: &mut Frame, area: Rect, splash: &SplashState, state: &AppState) {
    let theme = &state.theme;
    let lines = boot_lines(state);
    let elapsed = splash.mounted_at.elapsed();

    let visible = lines
        .iter()
        .take(splash.revealed.min(lines.len()))
        .enumerate()
        .map(|(idx, line)| {
            let ts = format!("[+{:.2}s]", (idx as f32 * 0.18).min(elapsed.as_secs_f32()));
            let glyph = status_glyph(&line.status, idx + 1 == lines.len(), splash);

            Line::from(vec![
                Span::styled(ts, Style::default().fg(theme.text_muted.to_ratatui())),
                Span::raw(" "),
                Span::styled("▸", Style::default().fg(theme.accent.to_ratatui())),
                Span::raw(" "),
                Span::styled(format!("{:<12}", line.phase), Style::default().fg(theme.text.to_ratatui())),
                Span::styled(format!("{:<16}", line.message), Style::default().fg(theme.text_muted.to_ratatui())),
                Span::styled(glyph, Style::default().fg(status_color(&line.status, state))),
            ])
        })
        .collect::<Vec<_>>();

    let height = visible.len() as u16;
    let y = area.y + area.height.saturating_sub(height + 1);
    let log_area = Rect {
        x: area.x + 2,
        y,
        width: area.width.saturating_sub(4),
        height: height.max(1),
    };

    frame.render_widget(Paragraph::new(visible), log_area);
}

fn render_brand(frame: &mut Frame, area: Rect, splash: &SplashState, state: &AppState) {
    let theme = &state.theme;
    let elapsed_ms = splash.mounted_at.elapsed().as_millis();
    let spinner = spinner_frame(splash);
    let pulse = pulse_style(elapsed_ms, state);

    let text = vec![
        Line::from(Span::styled(
            NEVER_HUMAN,
            Style::default()
                .fg(theme.text.to_ratatui())
                .add_modifier(Modifier::BOLD),
        )),
        Line::from(Span::styled(
            "▔".repeat(NEVER_HUMAN.chars().count()),
            Style::default().fg(theme.accent.to_ratatui()),
        )),
        Line::raw(""),
        Line::from(vec![
            Span::styled("Jekko ", Style::default().fg(theme.text.to_ratatui())),
            Span::styled("loading…", pulse),
        ]),
        Line::raw(""),
        Line::from(Span::styled("[q] quit", Style::default().fg(theme.text_muted.to_ratatui()))),
        Line::raw(""),
        Line::from(Span::styled(loading_band(area.width, spinner), pulse)),
    ];

    let h = text.len() as u16;
    let brand_area = Rect {
        x: area.x + 2,
        y: area.y + area.height.saturating_sub(h) / 2,
        width: area.width.saturating_sub(4),
        height: h,
    };

    frame.render_widget(Paragraph::new(text).alignment(Alignment::Center), brand_area);
}

fn boot_lines(state: &AppState) -> Vec<BootLine> {
    vec![
        BootLine { phase: "renderer", message: "ready".into(), status: BootStatus::Done },
        BootLine { phase: "terminal", message: format!("{}x{}", state.size.width, state.size.height), status: BootStatus::Done },
        BootLine { phase: "theme", message: state.theme_name.clone(), status: BootStatus::Done },
        BootLine { phase: "plugins", message: state.plugins.status_label(), status: state.plugins.boot_status() },
        BootLine { phase: "sync", message: state.sync.status_label(), status: state.sync.boot_status() },
        BootLine { phase: "jnoccio", message: state.jnoccio.status_label(), status: state.jnoccio.boot_status() },
        BootLine { phase: "jankurai", message: state.jankurai.status_label(), status: state.jankurai.boot_status() },
        BootLine { phase: "Jekko", message: if state.ready { "ready" } else { "waiting" }.into(), status: if state.ready { BootStatus::Done } else { BootStatus::Running } },
    ]
}

fn spinner_frame(splash: &SplashState) -> &'static str {
    let idx = (splash.mounted_at.elapsed().as_millis() / 90) as usize % SPINNER.len();
    SPINNER[idx]
}

fn status_glyph(status: &BootStatus, final_ready: bool, splash: &SplashState) -> &'static str {
    match status {
        BootStatus::Done if final_ready => "●",
        BootStatus::Done => "✓",
        BootStatus::Warn => "!",
        BootStatus::Error => "✗",
        BootStatus::Running => spinner_frame(splash),
    }
}

fn status_color(status: &BootStatus, state: &AppState) -> ratatui::style::Color {
    let theme = &state.theme;
    match status {
        BootStatus::Done => theme.success.to_ratatui(),
        BootStatus::Warn => theme.warning.to_ratatui(),
        BootStatus::Error => theme.error.to_ratatui(),
        BootStatus::Running => theme.accent.to_ratatui(),
    }
}

fn pulse_style(elapsed_ms: u128, state: &AppState) -> Style {
    let phase = elapsed_ms % PULSE_PERIOD_MS;
    let ratio = phase as f32 / PULSE_PERIOD_MS as f32;
    let mix = 0.5 + 0.5 * (ratio * std::f32::consts::TAU).sin();
    let color = state.theme.tint_accent_to_text(mix * 0.45);
    Style::default().fg(color.to_ratatui())
}

fn loading_band(width: u16, spinner: &str) -> String {
    let cells = ((width as f32 * 0.34) as usize).saturating_sub(4).max(12);
    (0..cells)
        .map(|i| match i % 6 {
            0 | 1 => spinner.repeat(2),
            2 | 4 => " ".to_string(),
            _ => spinner.to_string(),
        })
        .collect()
}
```

```text
width 80..99: left overlay width 28 only when visible
width 100..119: fixed left width 28
```

```rust
// crates/jekko-tui/src/screens/shell.rs

use ratatui::{
    Frame,
    layout::{Constraint, Direction, Layout, Rect},
    style::Style,
    widgets::Block,
};

use crate::state::AppState;

#[derive(Debug, Clone, Copy)]
pub enum LeftSize {
    Hidden,
    Shown { width: u16, overlay: bool },
}

#[derive(Default)]
pub struct ShellState {
    pub left_visible: bool,
}

pub fn resolve_left(width: u16, visible: bool) -> LeftSize {
    if width < 80 {
        LeftSize::Hidden
    } else if width < 100 {
        if visible {
            LeftSize::Shown { width: 28, overlay: true }
        } else {
            LeftSize::Hidden
        }
    } else if width < 120 {
        LeftSize::Shown { width: 28, overlay: false }
    } else if width < 160 {
        LeftSize::Shown { width: 38, overlay: false }
    } else {
        LeftSize::Shown { width: 44, overlay: false }
    }
}

pub fn render(frame: &mut Frame, area: Rect, state: &mut AppState) {
    let theme = &state.theme;

    let [body, footer] = Layout::default()
        .direction(Direction::Vertical)
        .constraints([Constraint::Min(0), Constraint::Length(3)])
        .split(area)
        .to_vec()
        .try_into()
        .unwrap();

    frame.render_widget(
        Block::default().style(Style::default().bg(theme.background.to_ratatui())),
        body,
    );

    match resolve_left(body.width, state.shell.left_visible) {
        LeftSize::Hidden => {
            render_center_feed(frame, body, state);
        }
        LeftSize::Shown { width, overlay: false } => {
            let chunks = Layout::default()
                .direction(Direction::Horizontal)
                .constraints([Constraint::Length(width), Constraint::Min(0)])
                .split(body);

            render_left_panel(frame, chunks[0], state);
            render_center_feed(frame, chunks[1], state);
        }
        LeftSize::Shown { width, overlay: true } => {
            render_center_feed(frame, body, state);

            let left = Rect {
                x: body.x,
                y: body.y,
                width: width.min(body.width),
                height: body.height,
            };

            frame.render_widget(ratatui::widgets::Clear, left);
            render_left_panel(frame, left, state);
        }
    }

    render_shell_footer(frame, footer, state);
}

fn render_left_panel(frame: &mut Frame, area: Rect, state: &mut AppState) {
    let theme = &state.theme;
    frame.render_widget(
        Block::default().style(Style::default().bg(theme.background_panel.to_ratatui())),
        area,
    );

    let [tabs, pane] = Layout::default()
        .direction(Direction::Vertical)
        .constraints([Constraint::Length(2), Constraint::Min(0)])
        .split(area)
        .to_vec()
        .try_into()
        .unwrap();

    state.plugins.render_slot(frame, tabs, "shell_left_tabs", theme);
    state.plugins.render_slot(frame, pane, "shell_left_pane", theme);
}

fn render_center_feed(frame: &mut Frame, area: Rect, state: &mut AppState) {
    if !state.plugins.render_slot(frame, area, "shell_activity_feed", &state.theme) {
        let paragraph = ratatui::widgets::Paragraph::new("No active feed")
            .style(state.theme.muted_style())
            .alignment(ratatui::layout::Alignment::Center);
        frame.render_widget(paragraph, area);
    }
}

fn render_shell_footer(frame: &mut Frame, area: Rect, state: &mut AppState) {
    crate::ui::footer_band::render(frame, area, &state.theme, |frame, area| {
        let line = ratatui::text::Line::from(vec![
            key(" Enter ", state), txt(" send  ", state),
            key(" Ctrl+P ", state), txt(" commands  ", state),
            key(" Tab ", state), txt(" pane  ", state),
            key(" Ctrl+B ", state), txt(" left  ", state),
            key(" Ctrl+Shift+T ", state), txt(" theme  ", state),
            key(" ? ", state), txt(" help  ", state),
            key(" Ctrl+C ", state), txt(" quit", state),

        frame.render_widget(ratatui::widgets::Paragraph::new(line), area);

}
```

```rust
// crates/jekko-tui/src/screens/session.rs

use ratatui::{
    Frame,
    layout::{Constraint, Direction, Layout, Rect},
    widgets::Block,
};

use crate::state::AppState;

#[derive(Default)]
pub struct SessionState {
    pub scroll: u16,
    pub sticky_bottom: bool,
    pub show_scrollbar: bool,
}

pub fn render(frame: &mut Frame, area: Rect, session_id: String, state: &mut AppState) {
    let theme = &state.theme;

    frame.render_widget(
        Block::default().style(ratatui::style::Style::default().bg(theme.background.to_ratatui())),
        area,
    );

    let padded = crate::ui::rect::inner(area, 2, 2, 0, 1);

    let bottom_height = bottom_stack_height(state, &session_id).min(padded.height);
    let [transcript_area, bottom_area] = Layout::default()
        .direction(Direction::Vertical)
        .constraints([
            Constraint::Min(0),
            Constraint::Length(bottom_height),
        ])
        .split(padded)
        .to_vec()
        .try_into()
        .unwrap();

    render_transcript(frame, transcript_area, &session_id, state);
    render_bottom_stack(frame, bottom_area, &session_id, state);
}

fn render_transcript(frame: &mut Frame, area: Rect, session_id: &str, state: &mut AppState) {
    let messages = state.sync.messages(session_id);

    let rendered = crate::session::renderers::render_messages(messages, state);
    let viewport = crate::ui::scroll::Viewport::new(area.height as usize, state.session.scroll as usize);

    let visible_lines = viewport.visible(rendered);
    frame.render_widget(ratatui::widgets::Paragraph::new(visible_lines), area);

    if state.session.show_scrollbar {
        crate::ui::scroll::render_scrollbar(frame, area, viewport, &state.theme);
    }
}

fn render_bottom_stack(frame: &mut Frame, area: Rect, session_id: &str, state: &mut AppState) {
    let mut rows = Vec::new();

    if state.sync.daemon_run(session_id).is_some() {
        rows.push(("daemon", 2));
    }
    if state.sync.permission(session_id).is_some() {
        rows.push(("permission", state.session_permission_height(session_id)));
    } else if state.sync.question(session_id).is_some() {
        rows.push(("question", state.session_question_height(session_id)));
    }
    if state.sync.is_subagent(session_id) {
        rows.push(("subagent", 1));
    }
    rows.push(("prompt", state.prompt.height()));

    let constraints = rows
        .iter()
        .map(|(_, h)| Constraint::Length(*h))
        .collect::<Vec<_>>();

    let chunks = Layout::default()
        .direction(Direction::Vertical)
        .constraints(constraints)
        .split(area);

    for ((name, _), rect) in rows.iter().zip(chunks.iter()) {
        match *name {
            "daemon" => crate::session::daemon_banner::render(frame, *rect, session_id, state),
            "permission" => crate::session::permission::render(frame, *rect, session_id, state),
            "question" => crate::session::question::render(frame, *rect, session_id, state),
            "subagent" => crate::session::subagent_footer::render(frame, *rect, session_id, state),
            "prompt" => crate::ui::prompt::render(frame, *rect, session_id, state),
            _ => {}
        }
    }
}

fn bottom_stack_height(state: &AppState, session_id: &str) -> u16 {
    let mut h = state.prompt.height();
    if state.sync.daemon_run(session_id).is_some() { h += 2; }
    if state.sync.permission(session_id).is_some() {
        h += state.session_permission_height(session_id);
    } else if state.sync.question(session_id).is_some() {
        h += state.session_question_height(session_id);
    }
    if state.sync.is_subagent(session_id) { h += 1; }
    h
}
```

### 10.3 Prompt event handling

```rust
// crates/jekko-tui/src/ui/prompt.rs

use crossterm::event::{KeyCode, KeyEvent, KeyModifiers};

use crate::{action::Action, state::AppState};

pub fn handle_key(key: KeyEvent, state: &mut AppState) -> Vec<Action> {
    if !state.prompt.focused || state.prompt.disabled {
        return vec![];
    }

    if state.prompt.autocomplete.visible {
        if let Some(action) = crate::ui::autocomplete::handle_key(key, state) {
            return vec![action];
        }
    }

    match key.code {
        KeyCode::Enter if !key.modifiers.contains(KeyModifiers::SHIFT) => {
            vec![Action::PromptSubmit]
        }
        KeyCode::Esc => {
            if state.sync.active_request_can_interrupt() {
                vec![Action::PromptInterrupt]
            } else {
                vec![Action::PromptBlur]
            }
        }
        KeyCode::Char('c') if key.modifiers.contains(KeyModifiers::CONTROL) => {
            handle_ctrl_c(state)
        }
        KeyCode::Char('p') if key.modifiers.contains(KeyModifiers::CONTROL) => {
            vec![Action::OpenCommandDialog]
        }
        KeyCode::Char(ch) => {
            state.prompt.textarea.input(ratatui_textarea::Input {
                key: ratatui_textarea::Key::Char(ch),
                ctrl: key.modifiers.contains(KeyModifiers::CONTROL),
                alt: key.modifiers.contains(KeyModifiers::ALT),
                shift: key.modifiers.contains(KeyModifiers::SHIFT),
            });
            state.prompt.recompute_mode_and_autocomplete();
            vec![Action::Redraw]
        }
        _ => {
            if state.prompt.textarea.input(key.into()) {
                state.prompt.recompute_mode_and_autocomplete();
                vec![Action::Redraw]
            } else {
                vec![]
            }
        }
    }
}

fn handle_ctrl_c(state: &mut AppState) -> Vec<Action> {
    let now = std::time::Instant::now();

    if state.sync.active_request_can_interrupt() {
        return vec![Action::PromptInterrupt];
    }

    if state
        .prompt
        .last_ctrl_c_at
        .is_some_and(|t| now.duration_since(t) < std::time::Duration::from_millis(900))
    {
        vec![Action::Quit]
    } else {
        state.prompt.last_ctrl_c_at = Some(now);
        vec![Action::ShowToast(crate::ui::toast::Toast::info(
            "Press Ctrl+C again to quit",
        ))]
    }
}
```

## 11. Dialog system migration

Current dialog system has a stack, escape handling, dialog sizes, select/prompt/confirm/alert variants, and mouse close behavior.

### 11.1 Dialog stack

```rust
// crates/jekko-tui/src/ui/dialog.rs

use ratatui::{
    Frame,
    layout::{Constraint, Direction, Layout, Rect},
    widgets::{Block, Borders, Clear},
};

use crate::state::AppState;

#[derive(Debug, Clone)]
pub enum Dialog {
    Alert(AlertDialog),
    Confirm(ConfirmDialog),
    Prompt(PromptDialog),
    Select(crate::ui::dialog_select::SelectDialog),
    Help(HelpDialog),
    Custom(crate::plugins::DeclarativeView),
}

#[derive(Default)]
pub struct DialogStack {
    stack: Vec<Dialog>,
}

impl DialogStack {
    pub fn replace(&mut self, dialog: Dialog) {
        self.stack.clear();
        self.stack.push(dialog);
    }

    pub fn push(&mut self, dialog: Dialog) {
        self.stack.push(dialog);
    }

    pub fn clear(&mut self) {
        self.stack.clear();
    }

    pub fn open(&self) -> bool {
        !self.stack.is_empty()
    }

    pub fn render(&mut self, frame: &mut Frame, screen: Rect, theme: &jekko_core::theme::ResolvedTheme) {
        let Some(dialog) = self.stack.last_mut() else {
            return;
        };

        let area = centered_rect(screen, dialog_width(screen), dialog_height(screen, dialog));
        frame.render_widget(Clear, area);

        let block = Block::default()
            .borders(Borders::ALL)
            .border_style(theme.border_style())
            .style(theme.panel_style());

        let inner = block.inner(area);
        frame.render_widget(block, area);

        match dialog {
            Dialog::Alert(d) => d.render(frame, inner, theme),
            Dialog::Confirm(d) => d.render(frame, inner, theme),
            Dialog::Prompt(d) => d.render(frame, inner, theme),
            Dialog::Select(d) => d.render(frame, inner, theme),
            Dialog::Help(d) => d.render(frame, inner, theme),
            Dialog::Custom(view) => view.render(frame, inner, theme),
        }
    }
}

fn centered_rect(screen: Rect, width: u16, height: u16) -> Rect {
    let width = width.min(screen.width.saturating_sub(4));
    let height = height.min(screen.height.saturating_sub(4));
    Rect {
        x: screen.x + screen.width.saturating_sub(width) / 2,
        y: screen.y + screen.height.saturating_sub(height) / 2,
        width,
        height,
    }
}

fn dialog_width(screen: Rect) -> u16 {
    match screen.width {
        0..=79 => screen.width.saturating_sub(4),
        80..=119 => 72,
        120..=179 => 88,
        _ => 104,
    }
}

fn dialog_height(screen: Rect, dialog: &Dialog) -> u16 {
    match dialog {
        Dialog::Select(d) => d.height(screen.height),
        Dialog::Prompt(_) => 8,
        Dialog::Confirm(_) => 8,
        Dialog::Alert(_) => 7,
        Dialog::Help(_) => screen.height.saturating_mul(3) / 4,
        Dialog::Custom(_) => screen.height.saturating_mul(2) / 3,
    }
}
```

```rust
    pub options: Vec<SelectOption>,
    pub render_filter: bool,
    pub flat_when_filtering: bool,
    pub keybinds: Vec<SelectKeybind>,
}

pub struct SelectKeybind {
    pub label: String,
    pub key: String,
    pub side: SelectKeybindSide,
}

pub enum SelectKeybindSide {
    Left,
    Right,
}

impl SelectDialog {
    pub fn height(&self, screen_height: u16) -> u16 {
        let rows = self.filtered_rows().len() as u16;
        let filter = if self.render_filter { 2 } else { 0 };
        let footer = if self.keybinds.is_empty() { 0 } else { 2 };
        rows.min(screen_height / 2).saturating_add(3 + filter + footer)
    }

    pub fn filtered_options(&self) -> Vec<&SelectOption> {
        let needle = self.filter.to_lowercase();
        let mut options = self
            .options
            .iter()
            .filter(|o| !o.disabled)
            .collect::<Vec<_>>();

        if needle.is_empty() {
            return options;
        }

        options.sort_by_key(|opt| fuzzy_score(&needle, opt).unwrap_or(i32::MAX));
        options
            .into_iter()
            .filter(|opt| fuzzy_score(&needle, opt).is_some())
            .collect()
    }

    pub fn render(
        &mut self,
        frame: &mut Frame,
        area: Rect,
        theme: &jekko_core::theme::ResolvedTheme,
    ) {
        let filter_height = if self.render_filter { 2 } else { 0 };
        let footer_height = if self.keybinds.is_empty() { 0 } else { 2 };

        let chunks = Layout::default()
            .direction(Direction::Vertical)
            .constraints([
                Constraint::Length(1),
                Constraint::Length(filter_height),
                Constraint::Min(1),
                Constraint::Length(footer_height),
            ])
            .split(area);

        render_title(frame, chunks[0], &self.title, theme);

        if self.render_filter {
            render_filter(frame, chunks[1], &self.filter, theme);
        }

        self.render_list(frame, chunks[2], theme);

        if footer_height > 0 {
            render_keybind_footer(frame, chunks[3], &self.keybinds, theme);
        }
    }

    fn render_list(
        &mut self,
        frame: &mut Frame,
        area: Rect,
        theme: &jekko_core::theme::ResolvedTheme,
    ) {
        let options = self.filtered_options();
        let items = options
            .iter()
            .map(|opt| {
                let marker = if opt.current { "● " } else { "  " };
                let mut spans = vec![
                    Span::styled(marker, Style::default().fg(theme.primary.to_ratatui())),
                    Span::styled(&opt.title, theme.text_style()),
                ];

                if let Some(desc) = &opt.description {
                    spans.push(Span::raw(" "));
                    spans.push(Span::styled(desc, theme.muted_style()));
                }

                if let Some(footer) = &opt.footer {
                    spans.push(Span::raw("  "));
                    spans.push(Span::styled(footer, theme.muted_style()));
                }

                ListItem::new(Line::from(spans))
            })
            .collect::<Vec<_>>();

        if items.is_empty() {
            frame.render_widget(
                Paragraph::new("No results found").style(theme.muted_style()),
                area,
            );
            return;
        }

        self.selected = self.selected.min(items.len().saturating_sub(1));

        let mut list_state = ListState::default();
        list_state.select(Some(self.selected));

        let list = List::new(items)
            .highlight_symbol(" ")
            .highlight_style(
                Style::default()
                    .fg(theme.selected_list_item_text.to_ratatui())
                    .bg(theme.primary.to_ratatui())
                    .add_modifier(Modifier::BOLD),
            );

        frame.render_stateful_widget(list, area, &mut list_state);
    }
}

fn render_title(
    frame: &mut Frame,
    area: Rect,
    title: &str,
    theme: &jekko_core::theme::ResolvedTheme,
) {
    let line = Line::from(vec![
        Span::styled(title.to_string(), theme.text_style().add_modifier(Modifier::BOLD)),
        Span::raw(" "),
        Span::styled("esc", theme.muted_style()),
    ]);
    frame.render_widget(Paragraph::new(line), area);
}

fn render_filter(
    frame: &mut Frame,
    area: Rect,
    filter: &str,
    theme: &jekko_core::theme::ResolvedTheme,
) {
    let line = Line::from(vec![
        Span::styled("Search ", theme.muted_style()),
        Span::styled(filter.to_string(), theme.text_style()),
    ]);
    frame.render_widget(Paragraph::new(line).style(theme.panel_style()), area);
}

fn render_keybind_footer(
    frame: &mut Frame,
    area: Rect,
    keybinds: &[SelectKeybind],
    theme: &jekko_core::theme::ResolvedTheme,
) {
    let spans = keybinds
        .iter()
        .flat_map(|kb| {
            [
                Span::styled(kb.label.clone(), theme.text_style().add_modifier(Modifier::BOLD)),
                Span::raw(" "),
                Span::styled(kb.key.clone(), theme.muted_style()),
                Span::raw("  "),
            ]
        })
        .collect::<Vec<_>>();

    frame.render_widget(Paragraph::new(Line::from(spans)), area);
}

fn fuzzy_score(needle: &str, option: &SelectOption) -> Option<i32> {
    let haystack = format!(
        "{} {}",
        option.title.to_lowercase(),
        option.category.clone().unwrap_or_default().to_lowercase()
    );

    let mut score = 0;
    let mut pos = 0usize;

    for ch in needle.chars() {
        let found = haystack[pos..].find(ch)?;
        score += found as i32;
        pos += found + ch.len_utf8();
    }

    Some(score)
}
```

```rust
use ratatui::{
    style::{Modifier, Style},
    text::{Line, Span},

use crate::state::AppState;

pub fn render_messages(
    messages: &[jekko_core::session::Message],
    state: &AppState,
) -> Vec<Line<'static>> {
    let mut lines = Vec::new();

    for (idx, message) in messages.iter().enumerate() {
        if idx > 0 {
            lines.push(Line::raw(""));
        }

        match message.role {
            jekko_core::session::Role::User => {
                render_user_message(&mut lines, message, state);
            }
            jekko_core::session::Role::Assistant => {
                render_assistant_message(&mut lines, message, state);
            }
        }
    }

    lines
}

fn render_user_message(
    lines: &mut Vec<Line<'static>>,
    message: &jekko_core::session::Message,
    state: &AppState,
) {
    let theme = &state.theme;
    let accent = state.agent_accent(&message.agent);

    lines.push(Line::from(vec![
        Span::styled("│ ", Style::default().fg(accent)),
        Span::styled("You", theme.text_style().add_modifier(Modifier::BOLD)),
        Span::styled(format_timestamp(message), theme.muted_style()),
    ]));

    for wrapped in crate::ui::text::wrap_lines(&message.content, state.content_width()) {
        lines.push(Line::from(vec![
            Span::styled("│ ", Style::default().fg(accent)),
            Span::styled(wrapped, theme.text_style()),
        ]));
    }

    for file in &message.files {
        lines.push(Line::from(vec![
            Span::styled("│ ", Style::default().fg(accent)),
            Span::styled(" @", theme.accent_style()),
            Span::styled(file.path.clone(), theme.muted_style()),
        ]));
    }
}

fn render_assistant_message(
    lines: &mut Vec<Line<'static>>,
    message: &jekko_core::session::Message,
    state: &AppState,
) {
    let theme = &state.theme;
    lines.push(Line::from(vec![
        Span::styled("▣ ", theme.accent_style()),
        Span::styled(message.model_label.clone(), theme.muted_style()),
        Span::raw(" "),
        Span::styled(format_duration(message.duration_ms), theme.muted_style()),
        Span::raw(" "),
        Span::styled(message.cost_label.clone().unwrap_or_default(), theme.muted_style()),
    ]));

    for part in &message.parts {
        match part {
            jekko_core::session::Part::Text(text) => {
                for wrapped in crate::ui::text::wrap_lines(text, state.content_width()) {
                    lines.push(Line::from(Span::styled(wrapped, theme.text_style())));
                }
            }
            jekko_core::session::Part::Reasoning(reasoning) => {
                render_reasoning(lines, reasoning, state);
            }
            jekko_core::session::Part::Tool(tool) => {
                crate::session::tools::render_tool(lines, tool, state);
            }
        }
    }
}

fn format_timestamp(message: &jekko_core::session::Message) -> Span<'static> {
    Span::raw(format!(" {}", message.time_label()))
}

fn format_duration(ms: Option<u64>) -> String {
    ms.map(|ms| format!("{:.1}s", ms as f64 / 1000.0)).unwrap_or_default()
}
```

## 13. Diff renderer replacement

Current OpenTUI uses a `<diff>` widget. Ratatui replacement should use `similar` and render either unified or split depending on width and config.

```rust
// crates/jekko-tui/src/ui/diff.rs

use ratatui::{
    style::Style,
    text::{Line, Span},
};

use similar::{ChangeTag, TextDiff};

use crate::state::AppState;

pub fn unified_diff_lines(
    old: &str,
    new: &str,
    state: &AppState,
) -> Vec<Line<'static>> {
    let theme = &state.theme;
    let diff = TextDiff::from_lines(old, new);
    let mut lines = Vec::new();

    for group in diff.grouped_ops(3) {
        lines.push(Line::from(Span::styled(
            "@@",
            Style::default()
                .fg(theme.diff_hunk_header.to_ratatui())
                .bg(theme.diff_context_bg.to_ratatui()),
        )));

        for op in group {
            for change in diff.iter_changes(&op) {
                let (prefix, fg, bg) = match change.tag() {
                    ChangeTag::Delete => ("-", theme.diff_removed, theme.diff_removed_bg),
                    ChangeTag::Insert => ("+", theme.diff_added, theme.diff_added_bg),
                    ChangeTag::Equal => (" ", theme.diff_context, theme.diff_context_bg),
                };

                lines.push(Line::from(vec![
                    Span::styled(
                        format!("{:>4} ", line_number_label(change.old_index())),
                        Style::default()
                            .fg(theme.diff_line_number.to_ratatui())
                            .bg(bg.to_ratatui()),
                    ),
                    Span::styled(
                        prefix,
                        Style::default().fg(fg.to_ratatui()).bg(bg.to_ratatui()),
                    ),
                    Span::styled(
                        change.value().trim_end_matches('\n').to_string(),
                        Style::default().fg(fg.to_ratatui()).bg(bg.to_ratatui()),
                    ),
                ]));
            }
        }
    }

    lines
}

fn line_number_label(index: Option<usize>) -> String {
    index.map(|i| (i + 1).to_string()).unwrap_or_else(|| "".into())
}
```

Split diff for wide screens:

```rust
pub fn split_diff_lines(old: &str, new: &str, width: u16, state: &AppState) -> Vec<Line<'static>> {
    if width < 120 {
        return unified_diff_lines(old, new, state);
    }

    let theme = &state.theme;
    let left_width = (width / 2).saturating_sub(2) as usize;
    let right_width = (width / 2).saturating_sub(2) as usize;
    let diff = TextDiff::from_lines(old, new);
    let mut lines = Vec::new();

    for change in diff.iter_all_changes() {
        match change.tag() {
            ChangeTag::Delete => {
                lines.push(Line::from(vec![
                    Span::styled(
                        crate::ui::text::truncate(change.value(), left_width),
                        Style::default()
                            .fg(theme.diff_removed.to_ratatui())
                            .bg(theme.diff_removed_bg.to_ratatui()),
                    ),
                    Span::raw(" │ "),
                    Span::raw(" ".repeat(right_width)),
                ]));
            }
            ChangeTag::Insert => {
                lines.push(Line::from(vec![
                    Span::raw(" ".repeat(left_width)),
                    Span::raw(" │ "),
                    Span::styled(
                        crate::ui::text::truncate(change.value(), right_width),
                        Style::default()
                            .fg(theme.diff_added.to_ratatui())
                            .bg(theme.diff_added_bg.to_ratatui()),
                    ),
                ]));
            }
            ChangeTag::Equal => {
                let text = change.value().trim_end_matches('\n');
                lines.push(Line::from(vec![
                    Span::styled(crate::ui::text::truncate(text, left_width), theme.muted_style()),
                    Span::raw(" │ "),
                    Span::styled(crate::ui::text::truncate(text, right_width), theme.muted_style()),
                ]));
            }
        }
    }

    lines
}
```

```rust
    Append,
    Replace,
    SingleWinner,
}

pub struct SlotContribution {
    pub plugin_id: String,
    pub priority: i32,
    pub view: jekko_plugin_api::View,
}

#[derive(Default)]
pub struct SlotRegistry {
    slots: BTreeMap<String, Vec<SlotContribution>>,
}

impl SlotRegistry {
    pub fn register(&mut self, name: impl Into<String>, contribution: SlotContribution) {
        self.slots.entry(name.into()).or_default().push(contribution);
    }

    pub fn render(
        &self,
        frame: &mut Frame,
        area: Rect,
        name: &str,
        mode: SlotMode,
        theme: &jekko_core::theme::ResolvedTheme,
    ) -> bool {
        let Some(items) = self.slots.get(name) else {
            return false;
        };

        match mode {
            SlotMode::Replace | SlotMode::SingleWinner => {
                if let Some(item) = items.iter().max_by_key(|item| item.priority) {
                    crate::plugins::view_renderer::render(frame, area, &item.view, theme);
                    true
                } else {
                    false
                }
            }
            SlotMode::Append => {
                let chunks = crate::ui::rect::vertical_even(area, items.len());
                for (item, chunk) in items.iter().zip(chunks) {
                    crate::plugins::view_renderer::render(frame, chunk, &item.view, theme);
                }
                true
            }
        }
    }
}
```

## 22. Recommended end state

The final result should have **no OpenTUI, no Bun, no Vite, and no TSX-rendered TUI**.

The Rust TUI should be organized like this:

```text
Ratatui immediate-mode renderer
Crossterm terminal/input backend
Tokio background services
Component/Action architecture
Theme tokens ported exactly
Prompt rebuilt around ratatui-textarea plus custom Jekko prompt parts
Dialogs rebuilt as Ratatui overlays
Session transcript rendered as styled line streams
Diffs rendered with similar
Plugins converted to Rust internal modules plus declarative external view models
CI/Nix/scripts moved to Cargo/xtask
```

The most important architectural decision is the plugin one: **external TSX/OpenTUI plugins must be treated as incompatible with this migration**. Preserve plugin behavior through a new declarative or Rust-native API, but do not keep the old TSX renderer hidden behind another JavaScript runtime. That is the only way to satisfy the “remove OpenTUI and Bun 100%” requirement cleanly.
