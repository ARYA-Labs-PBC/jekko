# Engineering Work Plan: Remove OpenTUI, Bun, and Vite; Replace UX/UI with Rust + Ratatui

## 1. Scope note from the uploaded archive

I inspected the uploaded archive as a focused “touchpoints” bundle. It contains **442 relevant non-AppleDouble files**, including **262 files under `packages/jekko/src/cli/cmd/tui`**, but it does **not** contain any `Cargo.toml` or Rust `.rs` files. So this plan is written as a migration plan from the provided TypeScript/OpenTUI/Bun TUI surface into a new or existing Rust workspace.

Important findings:

* **OpenTUI is deeply embedded** in the current TUI renderer, layout primitives, plugin API, test harness, build script, and theme/runtime types.
* **Bun is deeply embedded** in package management, scripts, tests, CI, Nix, TypeScript runtime code, `bun:sqlite`, `bun:ffi`, `Bun.stringWidth`, `Bun.file`, `Bun.write`, and `bun test`.
* **Solid is the current UI runtime** for the TUI. Removing OpenTUI also means removing Solid JSX from the TUI path.
* **Vite was not present in the uploaded archive.** I found zero Vite references in this touchpoint bundle. The full repo should still add a hard CI grep gate for `vite`, `Vite`, and `@vitejs/*`.

The target should be: **Rust binary owns all terminal UX/UI, Ratatui owns rendering, Crossterm owns terminal input/events, Cargo owns build/test/package, and no Bun/OpenTUI/Solid/Vite code remains in the CLI/TUI path.**

Ratatui is the right fit here because it is a Rust crate for building terminal UIs, and the current latest docs show `ratatui 0.30.0` as the current crate version. Ratatui’s own quickstart uses `ratatui::{DefaultTerminal, Frame}` plus `crossterm::event` for terminal input and drawing. ([Docs.rs][1]) Ratatui is immediate-mode: the application keeps state, handles events, and redraws the UI each iteration; Ratatui itself does not handle input, so Crossterm or another backend event source remains the correct pairing. ([Ratatui][2])

## 2. Executive migration decision

### Recommendation

Do **not** try to create a Rust wrapper around the existing OpenTUI/Solid app. That keeps the hard parts alive. Instead, create a Rust-native TUI crate and port the UI in layers:

1. Replace the OpenTUI renderer with a Ratatui terminal loop.
2. Replace Solid providers with a Rust `AppState`.
3. Replace JSX components with Ratatui render functions and stateful widgets.
4. Replace OpenTUI plugin slots with a Rust-native feature/slot registry.
5. Replace Bun tests/build/CI/Nix with Cargo, Rust integration tests, and Rust-friendly packaging.
6. Remove the TypeScript TUI tree after parity is reached.

### Canonical Rust/Ratatui architecture

Use a central app state plus actions/messages:

```text
Terminal events
    ↓
Event pump: crossterm/tokio
    ↓
Action enum
    ↓
App::update(action)
    ↓
Ratatui frame redraw
    ↓
App::render(frame)
```

This matches Ratatui’s model better than trying to recreate Solid’s reactive provider graph. The current TUI has many reactive providers, but in Rust these should become strongly typed state structs and focused modules.

Ratatui 0.30.0 also reorganized the project into a modular workspace, but the Ratatui migration notes explicitly say regular application developers can continue using the main `ratatui` crate; that is what I recommend for this app. ([Ratatui][3])

### 7.2 `crates/jekko-tui/src/tui.rs`

This replaces OpenTUI renderer creation, keyboard hooks, frame callbacks, and terminal cleanup.

```rust
use std::{io, time::Duration};

use color_eyre::eyre::Result;
use crossterm::{
    cursor,
    event::{
        DisableBracketedPaste, DisableFocusChange, DisableMouseCapture,
        EnableBracketedPaste, EnableFocusChange, EnableMouseCapture,
        Event as CrosstermEvent, EventStream, KeyEventKind,
    },
    execute,
    terminal::{EnterAlternateScreen, LeaveAlternateScreen},
};
use futures::{FutureExt, StreamExt};
use ratatui::{backend::CrosstermBackend, Terminal};
use tokio::{
    sync::mpsc::{self, UnboundedReceiver, UnboundedSender},
    task::JoinHandle,
};
use tokio_util::sync::CancellationToken;

use crate::action::Action;

pub type Backend = CrosstermBackend<io::Stdout>;
pub type TuiTerminal = Terminal<Backend>;

pub struct Tui {
    terminal: TuiTerminal,
    rx: UnboundedReceiver<Action>,
    tx: UnboundedSender<Action>,
    task: Option<JoinHandle<()>>,
    cancel: CancellationToken,
    mouse: bool,
    paste: bool,
    focus: bool,
    frame_rate: f64,
    tick_rate: f64,
}

impl Tui {
    pub fn new(mouse: bool, paste: bool) -> Result<Self> {
        let backend = CrosstermBackend::new(io::stdout());
        let terminal = Terminal::new(backend)?;
        let (tx, rx) = mpsc::unbounded_channel();

        Ok(Self {
            terminal,
            rx,
            tx,
            task: None,
            cancel: CancellationToken::new(),
            mouse,
            paste,
            focus: true,
            frame_rate: 60.0,
            tick_rate: 4.0,
        })
    }

    pub fn enter(&mut self) -> Result<()> {
        crossterm::terminal::enable_raw_mode()?;
        execute!(io::stdout(), EnterAlternateScreen, cursor::Hide)?;

        if self.mouse {
            execute!(io::stdout(), EnableMouseCapture)?;
        }
        if self.focus {
            execute!(io::stdout(), EnableFocusChange)?;
        }
        if self.paste {
            execute!(io::stdout(), EnableBracketedPaste)?;
        }

        self.start_event_task();
        Ok(())
    }

    pub fn exit(&mut self) -> Result<()> {
        self.cancel.cancel();

        if let Some(task) = self.task.take() {
            task.abort();
        }

        if self.paste {
            let _ = execute!(io::stdout(), DisableBracketedPaste);
        }
        if self.focus {
            let _ = execute!(io::stdout(), DisableFocusChange);
        }
        if self.mouse {
            let _ = execute!(io::stdout(), DisableMouseCapture);
        }

        let _ = execute!(io::stdout(), LeaveAlternateScreen, cursor::Show);
        let _ = crossterm::terminal::disable_raw_mode();
        Ok(())
    }

    pub fn draw<F>(&mut self, f: F) -> Result<()>
    where
        F: FnOnce(&mut ratatui::Frame),
    {
        self.terminal.draw(f)?;
        Ok(())
    }

    pub async fn next(&mut self) -> Option<Action> {
        self.rx.recv().await
    }

    fn start_event_task(&mut self) {
        self.cancel = CancellationToken::new();
        let cancel = self.cancel.clone();
        let tx = self.tx.clone();
        let frame_delay = Duration::from_secs_f64(1.0 / self.frame_rate);
        let tick_delay = Duration::from_secs_f64(1.0 / self.tick_rate);

        self.task = Some(tokio::spawn(async move {
            let mut reader = EventStream::new();
            let mut frame = tokio::time::interval(frame_delay);
            let mut tick = tokio::time::interval(tick_delay);

            loop {
                let next_event = reader.next().fuse();

                tokio::select! {
                    _ = cancel.cancelled() => break,

                    _ = frame.tick() => {
                        let _ = tx.send(Action::Render);
                    }

                    _ = tick.tick() => {
                        let _ = tx.send(Action::Tick);
                    }

                    maybe_event = next_event => {
                        match maybe_event {
                            Some(Ok(CrosstermEvent::Key(key))) if key.kind == KeyEventKind::Press => {
                                let _ = tx.send(Action::Key(key));
                            }
                            Some(Ok(CrosstermEvent::Mouse(mouse))) => {
                                let _ = tx.send(Action::Mouse(mouse));
                            }
                            Some(Ok(CrosstermEvent::Resize(width, height))) => {
                                let _ = tx.send(Action::Resize { width, height });
                            }
                            Some(Ok(CrosstermEvent::Paste(text))) => {
                                let _ = tx.send(Action::Paste(text));
                            }
                            Some(Ok(_)) => {}
                            Some(Err(_)) => {
                                let _ = tx.send(Action::ForceRedraw);
                            }
                            None => {}
                        }
                    }
                }
            }
        }));
    }
}

impl Drop for Tui {
    fn drop(&mut self) {
        let _ = self.exit();
    }
}
```

This mirrors the Ratatui recipe pattern of a terminal wrapper with Crossterm `EventStream`, Tokio `select!`, tick/render events, mouse, paste, raw mode, and alternate screen. ([Ratatui][5])

### 7.3 `crates/jekko-tui/src/app.rs`

```rust
use color_eyre::eyre::Result;
use ratatui::Frame;

use crate::{
    action::{Action, Route},
    components::{dialog::DialogStack, toast::ToastState},
    config::tui::TuiConfig,
    state::AppState,
    tui::Tui,
};

pub struct App {
    pub state: AppState,
    pub dialogs: DialogStack,
    pub toast: ToastState,
    should_quit: bool,
}

impl App {
    pub async fn new(config: TuiConfig) -> Result<Self> {
        Ok(Self {
            state: AppState::new(config).await?,
            dialogs: DialogStack::default(),
            toast: ToastState::default(),
            should_quit: false,
        })
    }

    pub async fn run(mut self) -> Result<()> {
        let mouse = self.state.config.mouse.unwrap_or(true);
        let mut tui = Tui::new(mouse, true)?;
        tui.enter()?;

        while !self.should_quit {
            tui.draw(|frame| self.render(frame))?;

            if let Some(action) = tui.next().await {
                self.update(action).await?;
            }
        }

        tui.exit()?;
        Ok(())
    }

    pub fn render(&mut self, frame: &mut Frame) {
        crate::routes::render(frame, &mut self.state, &mut self.dialogs, &mut self.toast);
    }

    pub async fn update(&mut self, action: Action) -> Result<()> {
        if self.dialogs.handle_action(&action, &mut self.state).await? {
            return Ok(());
        }

        match action {
            Action::Quit => self.should_quit = true,

            Action::Navigate(route) => self.state.route.navigate(route),
            Action::NavigateBack => self.state.route.navigate_back(),

            Action::OpenHelp => self.dialogs.help(),
            Action::OpenStatus => self.dialogs.status(),
            Action::OpenCommandDialog => self.dialogs.command_palette(&self.state),

            Action::ToggleThemeMode => self.state.theme.toggle_mode(),
            Action::ToggleShellLeft => self.state.local.shell_left_visible = !self.state.local.shell_left_visible,

            Action::NewSession => {
                self.state.route.navigate(Route::Shell);
                self.dialogs.clear();
                self.state.prompt.reset();
            }

            Action::ResumeSession => {
                if let Some(session_id) = self.state.sync.most_recent_root_session() {
                    self.state.route.navigate(Route::Session { session_id });
                } else {
                    self.toast.info("No previous session to resume");
                }
            }

            Action::Key(key) => {
                let maybe = self.state.keybinds.resolve(key, self.state.prompt.focused);
                if let Some(action) = maybe {
                    Box::pin(self.update(action)).await?;
                } else {
                    self.state.prompt.handle_key(key)?;
                }
            }

            Action::Paste(text) => self.state.prompt.insert_str(&text),

            Action::Resize { width, height } => {
                self.state.terminal.width = width;
                self.state.terminal.height = height;
            }

            Action::Tick => {
                self.toast.tick();
                self.state.animations.tick();
            }

            Action::Render | Action::ForceRedraw => {}

            other => self.state.update(other).await?,
        }

        Ok(())
    }
}
```

```rust
    layout::{Constraint, Layout},
    prelude::*,
    widgets::Clear,

use crate::{
    action::Route,
```

## 12. Logo port: preserving the Jekko look

The current `logo.tsx` is custom and brand-critical:

* `INNER_WIDTH = 78`, `OUTER_WIDTH = 80`.
* Amber palette anchored around canonical `#D4A843`.
* Dark amber stops and light amber stops.
* Wordmark/shadow layers.
* Gradient lookup table.
* Contrast lifting on black.
* Pixel-font/wordmark data.

Do **not** replace it with plain text or a generic FIGlet. Port the logic.

### `components/logo.rs`

```rust
use ratatui::{prelude::*, widgets::Paragraph};

use crate::state::AppState;

pub const INNER_WIDTH: u16 = 78;
pub const OUTER_WIDTH: u16 = INNER_WIDTH + 2;

#[derive(Debug, Clone, Copy)]
pub enum LogoMode {
    Idle,
    Active,
}

#[derive(Debug, Clone, Copy)]
pub struct Rgb {
    r: u8,
    g: u8,
    b: u8,
}

#[derive(Debug, Clone)]
pub enum CellLayer {
    Global,
    Wordmark,
    ShadowNear,
    ShadowMid,
    ShadowFar,
}

#[derive(Debug, Clone)]
pub struct LogoCell {
    pub ch: char,
    pub layer: CellLayer,
    pub strong: bool,
    pub dim: bool,
    pub source_x: u16,
    pub source_y: u16,
    pub source_width: u16,
    pub source_height: u16,
}

#[derive(Debug, Clone)]
pub struct LogoRow {
    pub cells: Vec<LogoCell>,
}

const AMBER_DARK: &[&str] = &[
    "#3D2606",
    "#5C3A0E",
    "#8B5F1A",
    "#B57F28",
    "#D4A843",
    "#E8C055",
    "#EFD17A",
    "#F8E3B3",
];

const AMBER_LIGHT: &[&str] = &[
    "#2D1A04",
    "#4F3408",
    "#553707",
    "#6E4A0A",
    "#7C5A11",
    "#8C5F0D",
    "#A67817",
    "#B98828",
];

pub fn height() -> u16 {
    logo_rows().len() as u16
}

pub fn render(frame: &mut Frame, area: Rect, state: &AppState, _mode: LogoMode) {
    let rows = logo_rows();
    let x = match area.width.cmp(&OUTER_WIDTH) {
        std::cmp::Ordering::Greater => area.x + (area.width - OUTER_WIDTH) / 2,
        _ => area.x,
    };

    let palette = if state.theme.mode.is_dark() {
        build_gradient(AMBER_DARK, 512)
    } else {
        build_gradient(AMBER_LIGHT, 512)
    };

    let mut lines = Vec::with_capacity(rows.len());

    for row in rows {
        let mut spans = Vec::with_capacity(row.cells.len());
        for cell in row.cells {
            let color = color_for_cell(&cell, &palette);
            let mut style = Style::default().fg(color);

            if cell.strong {
                style = style.add_modifier(Modifier::BOLD);
            }
            if cell.dim {
                style = style.add_modifier(Modifier::DIM);
            }

            spans.push(Span::styled(cell.ch.to_string(), style));
        }
        lines.push(Line::from(spans));
    }

    let target = Rect {
        x,
        y: area.y,
        width: OUTER_WIDTH.min(area.width),
        height: height().min(area.height),
    };

    frame.render_widget(Paragraph::new(lines), target);
}

fn color_for_cell(cell: &LogoCell, palette: &[Color]) -> Color {
    if palette.is_empty() {
        return Color::Rgb(212, 168, 67);
    }

    let denom = cell.source_width.max(1) as f32;
    let t = (cell.source_x as f32 / denom).clamp(0.0, 1.0);
    let idx = ((palette.len() - 1) as f32 * t).round() as usize;

    match cell.layer {
        CellLayer::ShadowFar => darken(palette[idx], 0.55),
        CellLayer::ShadowMid => darken(palette[idx], 0.40),
        CellLayer::ShadowNear => darken(palette[idx], 0.25),
        CellLayer::Global | CellLayer::Wordmark => palette[idx],
    }
}

fn build_gradient(stops: &[&str], steps: usize) -> Vec<Color> {
    let stops: Vec<Rgb> = stops.iter().map(|s| parse_rgb(s)).collect();
    let mut out = Vec::with_capacity(steps);

    for i in 0..steps {
        let t = i as f32 / (steps.saturating_sub(1).max(1)) as f32;
        let scaled = t * (stops.len().saturating_sub(1)) as f32;
        let left = scaled.floor() as usize;
        let right = (left + 1).min(stops.len() - 1);
        let local = scaled - left as f32;
        let rgb = mix(stops[left], stops[right], smoothstep(local));
        out.push(Color::Rgb(rgb.r, rgb.g, rgb.b));
    }

    out
}

fn parse_rgb(hex: &str) -> Rgb {
    let clean = hex.trim_start_matches('#');
    let r = u8::from_str_radix(&clean[0..2], 16).unwrap_or(255);
    let g = u8::from_str_radix(&clean[2..4], 16).unwrap_or(0);
    let b = u8::from_str_radix(&clean[4..6], 16).unwrap_or(184);
    Rgb { r, g, b }
}

fn mix(a: Rgb, b: Rgb, t: f32) -> Rgb {
    let k = t.clamp(0.0, 1.0);
    Rgb {
        r: (a.r as f32 + (b.r as f32 - a.r as f32) * k).round() as u8,
        g: (a.g as f32 + (b.g as f32 - a.g as f32) * k).round() as u8,
        b: (a.b as f32 + (b.b as f32 - a.b as f32) * k).round() as u8,
    }
}

fn smoothstep(t: f32) -> f32 {
    let x = t.clamp(0.0, 1.0);
    x * x * (3.0 - 2.0 * x)
}

fn darken(color: Color, amount: f32) -> Color {
    match color {
        Color::Rgb(r, g, b) => {
            let k = (1.0 - amount).clamp(0.0, 1.0);
            Color::Rgb(
                (r as f32 * k).round() as u8,
                (g as f32 * k).round() as u8,
                (b as f32 * k).round() as u8,
            )
        }
        other => other,
    }
}
```

```rust
#[derive(Debug, Clone, Copy, PartialEq, Eq, Serialize, Deserialize)]
pub struct Rgba {
    pub r: u8,
    pub g: u8,
    pub b: u8,
    #[serde(default = "opaque")]
    pub a: u8,
}

fn opaque() -> u8 {
    255
}

impl Rgba {
    pub fn transparent() -> Self {
        Self { r: 0, g: 0, b: 0, a: 0 }
    }

    pub fn to_ratatui(self) -> Color {
        if self.a == 0 {
            Color::Reset
        } else {
            Color::Rgb(self.r, self.g, self.b)
        }
    }

    pub fn from_hex(input: &str) -> Result<Self> {
        let clean = input.trim().trim_start_matches('#');
        if clean.len() != 6 && clean.len() != 8 {
            return Err(eyre!("expected #RRGGBB or #RRGGBBAA, got {input}"));
        }

        let r = u8::from_str_radix(&clean[0..2], 16)?;
        let g = u8::from_str_radix(&clean[2..4], 16)?;
        let b = u8::from_str_radix(&clean[4..6], 16)?;
        let a = if clean.len() == 8 {
            u8::from_str_radix(&clean[6..8], 16)?
        } else {
            255
        };

        Ok(Self { r, g, b, a })
    }
```

```rust
#[derive(Debug, Clone, Copy, PartialEq, Eq)]

    Dark,
    Light,
}

impl ThemeMode {
    pub fn is_dark(self) -> bool {

    }

pub fn resolve(json: &ThemeJson, mode: ThemeMode) -> Result<Theme> {
    let color = |value: &ColorValue| -> Result<Color> {
        resolve_color(json, value, mode, &mut Vec::new()).map(Rgba::to_ratatui)
    };

    Ok(Theme {
        primary: color(&json.theme.primary)?,
        secondary: color(&json.theme.secondary)?,

        info: color(&json.theme.info)?,

        text: color(&json.theme.text)?,

        background_panel: color(&json.theme.background_panel)?,
        background_element: color(&json.theme.background_element)?,
        background_menu: match &json.theme.background_menu {
            Some(value) => color(value)?,
            None => color(&json.theme.background_element)?,
        },

        border: color(&json.theme.border)?,
        border_active: color(&json.theme.border_active)?,
        border_subtle: color(&json.theme.border_subtle)?,

        thinking_opacity: json.theme.thinking_opacity,
    })
}

fn resolve_color(
    json: &ThemeJson,
    value: &ColorValue,
    mode: ThemeMode,
    chain: &mut Vec<String>,
) -> Result<Rgba> {
    match value {
        ColorValue::Rgba(rgba) => Ok(*rgba),
        ColorValue::Ansi(code) => Ok(ansi_to_rgba(*code)),
        ColorValue::Hex(s) if s == "transparent" || s == "none" => Ok(Rgba::transparent()),
        ColorValue::Hex(s) if s.starts_with('#') => Rgba::from_hex(s),
        ColorValue::Ref(name) | ColorValue::Hex(name) => {
            if chain.contains(name) {
                return Err(eyre!("circular theme reference: {:?}", chain));
            }

            let Some(next) = json.defs.get(name) else {
                return Err(eyre!("theme reference not found: {name}"));
            };

            chain.push(name.clone());
            let out = resolve_color(json, next, mode, chain)?;
            chain.pop();
            Ok(out)
        }
        ColorValue::Variant { dark, light } => {
            let next = if mode == ThemeMode::Dark { dark } else { light };
            resolve_color(json, next, mode, chain)
        }
    }
}

fn ansi_to_rgba(code: u8) -> Rgba {
    const ANSI: [&str; 16] = [
        "#000000", "#800000", "#008000", "#808000",
        "#000080", "#800080", "#008080", "#c0c0c0",
        "#808080", "#ff0000", "#00ff00", "#ffff00",
        "#0000ff", "#ff00ff", "#00ffff", "#ffffff",
    ];

    Rgba::from_hex(ANSI.get(code as usize).copied().unwrap_or("#000000"))
        .unwrap_or(Rgba { r: 0, g: 0, b: 0, a: 255 })
}
```

```rust
use std::{collections::HashMap, time::{Duration, Instant}};

use crossterm::event::{KeyCode, KeyEvent, KeyModifiers};
use serde::{Deserialize, Serialize};

use crate::action::Action;

#[derive(Debug, Clone, PartialEq, Eq, Hash, Serialize, Deserialize)]
pub struct Keybind {
    pub name: String,
    pub ctrl: bool,
    pub alt: bool,
    pub shift: bool,
    pub super_key: bool,
    pub leader: bool,
}

#[derive(Debug)]
pub struct KeybindState {
    pub bindings: HashMap<String, Vec<Keybind>>,
    leader_active_since: Option<Instant>,
}

impl KeybindState {
    pub fn new(bindings: HashMap<String, Vec<Keybind>>) -> Self {
        Self { bindings, leader_active_since: None }
    }

    pub fn parse_all(raw: &HashMap<String, String>) -> HashMap<String, Vec<Keybind>> {
        raw.iter()
            .map(|(name, value)| (name.clone(), parse_combo_list(value)))
            .collect()
    }

    pub fn resolve(&mut self, key: KeyEvent, text_input_focused: bool) -> Option<Action> {
        self.expire_leader();

        if self.matches_name("leader", key) {
            self.leader_active_since = Some(Instant::now());
            return None;
        }

        let sensitive = matches!(
            key_name(&key).as_deref(),
            Some("?") | Some("g") | Some("y") | Some("r") | Some("1") | Some("2") | Some("3")
        );

        let leader = self.leader_active_since.is_some();

        for (name, list) in &self.bindings {
            if text_input_focused && sensitive && is_text_input_sensitive(name) {
                continue;
            }

            if list.iter().any(|b| b.matches(key, leader)) {
                self.leader_active_since = None;
                return binding_to_action(name, key);
            }
        }

        if leader {
            self.leader_active_since = None;
        }

        None
    }

    fn matches_name(&self, name: &str, key: KeyEvent) -> bool {
        self.bindings
            .get(name)
            .map(|list| list.iter().any(|b| b.matches(key, false)))
            .unwrap_or(false)
    }

    fn expire_leader(&mut self) {
        if self
            .leader_active_since
            .is_some_and(|started| started.elapsed() > Duration::from_secs(2))
        {
            self.leader_active_since = None;
        }
    }
}

impl Keybind {
    pub fn matches(&self, key: KeyEvent, leader_active: bool) -> bool {
        self.leader == leader_active
            && self.ctrl == key.modifiers.contains(KeyModifiers::CONTROL)
            && self.alt == key.modifiers.contains(KeyModifiers::ALT)
            && self.shift == key.modifiers.contains(KeyModifiers::SHIFT)
            && self.name == key_name(&key).unwrap_or_default()
    }
}

pub fn parse_combo_list(input: &str) -> Vec<Keybind> {
    if input == "none" {
        return vec![];
    }

    input
        .split(',')
        .filter_map(|combo| parse_one(combo.trim()))
        .collect()
}

fn parse_one(combo: &str) -> Option<Keybind> {
    let normalized = combo.replace("<leader>", "leader+");
    let mut out = Keybind {
        name: String::new(),
        ctrl: false,
        alt: false,
        shift: false,
        super_key: false,
        leader: false,
    };

    for part in normalized.to_lowercase().split('+') {
        match part {
            "ctrl" => out.ctrl = true,
            "alt" | "meta" | "option" => out.alt = true,
            "shift" => out.shift = true,
            "super" => out.super_key = true,
            "leader" => out.leader = true,
            "esc" => out.name = "escape".into(),
            "space" => out.name = "space".into(),
            "" => {}
            other => out.name = other.into(),
        }
    }

    if out.name.is_empty() && !out.leader {
        None
    } else {
        Some(out)
    }
}

fn key_name(key: &KeyEvent) -> Option<String> {
    match key.code {
        KeyCode::Char(' ') => Some("space".into()),
        KeyCode::Char(c) => Some(c.to_ascii_lowercase().to_string()),
        KeyCode::Enter => Some("enter".into()),
        KeyCode::Esc => Some("escape".into()),
        KeyCode::Backspace => Some("backspace".into()),
        KeyCode::Delete => Some("delete".into()),
        KeyCode::Tab => Some("tab".into()),
        KeyCode::Up => Some("up".into()),
        KeyCode::Down => Some("down".into()),
        KeyCode::Left => Some("left".into()),
        KeyCode::Right => Some("right".into()),
        _ => None,
    }
}

fn is_text_input_sensitive(name: &str) -> bool {
    matches!(
        name,
        "help.show"
            | "feed.scroll.top"
            | "feed.scroll.bottom"
            | "shell.tab.set"
            | "engage"
    )
}

fn binding_to_action(name: &str, key: KeyEvent) -> Option<Action> {
    match name {
        "engage" => Some(Action::Navigate(crate::action::Route::Shell)),
        "shell.left.toggle" => Some(Action::ToggleShellLeft),
        "theme.mode.toggle" => Some(Action::ToggleThemeMode),
        "session.new" => Some(Action::NewSession),
        "session.resume" => Some(Action::ResumeSession),
        "help.show" => Some(Action::OpenHelp),
        "feed.scroll.pageUp" => Some(Action::ScrollPageUp),
        "feed.scroll.pageDown" => Some(Action::ScrollPageDown),
        "feed.scroll.top" => Some(Action::ScrollTop),
        "feed.scroll.bottom" => Some(Action::ScrollBottom),
        "shell.tab.cycle" => Some(Action::CycleShellTab { reverse: false }),
        "shell.tab.cycleBack" => Some(Action::CycleShellTab { reverse: true }),
        "shell.tab.set" => Some(Action::SetShellTab(match key.code {
            KeyCode::Char(c) => c.to_string(),
            _ => return None,
        })),
        _ => None,
    }
}
```

```rust
use crossterm::event::{KeyCode, KeyEvent, KeyModifiers};
use ratatui::{prelude::*, widgets::{Block, Borders, Paragraph}};
use unicode_segmentation::UnicodeSegmentation;
use unicode_width::UnicodeWidthStr;

#[derive(Debug, Default, Clone)]
pub struct TextAreaState {
    text: String,
    cursor_grapheme: usize,
    focused: bool,
}

impl TextAreaState {
    pub fn text(&self) -> &str {
        &self.text
    }

    pub fn set(&mut self, value: impl Into<String>) {
        self.text = value.into();
        self.cursor_grapheme = self.grapheme_count();
    }

    pub fn insert_str(&mut self, value: &str) {
        let byte = self.cursor_byte();
        self.text.insert_str(byte, value);
        self.cursor_grapheme += value.graphemes(true).count();
    }

    pub fn handle_key(&mut self, key: KeyEvent) -> bool {
        match key.code {
            KeyCode::Char(c) if !key.modifiers.intersects(KeyModifiers::CONTROL | KeyModifiers::ALT) => {
                let byte = self.cursor_byte();
                self.text.insert(byte, c);
                self.cursor_grapheme += 1;
                true
            }
            KeyCode::Backspace => {
                if self.cursor_grapheme == 0 {
                    return false;
                }

                let start = self.grapheme_byte(self.cursor_grapheme - 1);
                let end = self.grapheme_byte(self.cursor_grapheme);
                self.text.replace_range(start..end, "");
                self.cursor_grapheme -= 1;
                true
            }
            KeyCode::Delete => {
                if self.cursor_grapheme >= self.grapheme_count() {
                    return false;
                }

                let start = self.grapheme_byte(self.cursor_grapheme);
                let end = self.grapheme_byte(self.cursor_grapheme + 1);
                self.text.replace_range(start..end, "");
                true
            }
            KeyCode::Left => {
                self.cursor_grapheme = self.cursor_grapheme.saturating_sub(1);
                true
            }
            KeyCode::Right => {
                self.cursor_grapheme = (self.cursor_grapheme + 1).min(self.grapheme_count());
                true
            }
            KeyCode::Home => {
                self.cursor_grapheme = 0;
                true
            }
            KeyCode::End => {
                self.cursor_grapheme = self.grapheme_count();
                true
            }
            _ => false,
        }
    }

    pub fn render(&self, frame: &mut Frame, area: Rect, theme: &crate::theme::Theme) {
        let block = Block::default()
            .borders(Borders::ALL)
            .border_style(Style::default().fg(if self.focused { theme.border_active } else { theme.border }));

        let inner = block.inner(area);
        frame.render_widget(block, area);

        frame.render_widget(
            Paragraph::new(self.text.as_str())
                .style(Style::default().fg(theme.text))
                .wrap(ratatui::widgets::Wrap { trim: false }),
            inner,
        );

        if self.focused {
            let before = self.before_cursor();
            let cursor_x = before.width() as u16;
            frame.set_cursor_position(Position {
                x: inner.x + cursor_x.min(inner.width.saturating_sub(1)),
                y: inner.y,
            });
        }
    }

    fn before_cursor(&self) -> &str {
        &self.text[..self.cursor_byte()]
    }

    fn cursor_byte(&self) -> usize {
        self.grapheme_byte(self.cursor_grapheme)
    }

    fn grapheme_count(&self) -> usize {
        self.text.graphemes(true).count()
    }

    fn grapheme_byte(&self, idx: usize) -> usize {
        self.text
            .grapheme_indices(true)
            .nth(idx)
            .map(|(byte, _)| byte)
            .unwrap_or(self.text.len())
    }
}
```

```rust
use color_eyre::eyre::Result;
use ratatui::{
    layout::{Constraint, Layout, Rect},
    prelude::*,
    widgets::{Block, BorderType, Borders, Clear, List, ListItem, Paragraph},
};

use crate::{action::Action, state::AppState};

#[derive(Debug, Clone, Copy)]
pub enum DialogSize {
    Medium,
    Large,
    XLarge,
}

#[derive(Debug)]
pub enum Dialog {
    Help,
    Status,
    CommandPalette(CommandPaletteState),
    Alert { title: String, message: String },
    Confirm { title: String, message: String, command: ConfirmCommand },
    Select(SelectState),
    Prompt(PromptDialogState),
}

#[derive(Debug, Default)]
pub struct DialogStack {
    stack: Vec<Dialog>,
    size: DialogSize,
}

impl Default for DialogSize {
    fn default() -> Self {
        Self::Medium
    }
}

impl DialogStack {
    pub fn is_open(&self) -> bool {
        !self.stack.is_empty()
    }

    pub fn clear(&mut self) {
        self.stack.clear();
    }

    pub fn replace(&mut self, dialog: Dialog) {
        self.stack.clear();
        self.stack.push(dialog);
    }

    pub fn help(&mut self) {
        self.replace(Dialog::Help);
    }

    pub fn status(&mut self) {
        self.replace(Dialog::Status);
    }

    pub async fn handle_action(&mut self, action: &Action, state: &mut AppState) -> Result<bool> {
        if self.stack.is_empty() {
            return Ok(false);
        }

        match action {
            Action::Key(key) => {
                if crate::util::terminal::is_escape(*key) {
                    self.clear();
                    return Ok(true);
                }

                if let Some(dialog) = self.stack.last_mut() {
                    return dialog.handle_key(*key, state).await;
                }

                Ok(true)
            }
            _ => Ok(false),
        }
    }

    pub fn render(&mut self, frame: &mut Frame, area: Rect, state: &AppState) {
        let Some(dialog) = self.stack.last_mut() else {
            return;
        };

        let rect = centered_rect(area, self.size);
        frame.render_widget(Clear, rect);

        let block = Block::default()
            .borders(Borders::ALL)
            .border_type(BorderType::Rounded)
            .style(Style::default().bg(state.theme.current.background_panel))
            .border_style(Style::default().fg(state.theme.current.border_active));

        let inner = block.inner(rect);
        frame.render_widget(block, rect);

        dialog.render(frame, inner, state);
    }
}

fn centered_rect(area: Rect, size: DialogSize) -> Rect {
    let (w, h) = match size {
        DialogSize::Medium => (60, 16),
        DialogSize::Large => (84, 24),
        DialogSize::XLarge => (110, 32),
    };

    Rect {
        x: area.x + area.width.saturating_sub(w) / 2,
        y: area.y + area.height.saturating_sub(h) / 2,
        width: w.min(area.width),
        height: h.min(area.height),
    }
}
```

```rust
use ratatui::{
    layout::{Constraint, Layout, Rect},
    prelude::*,
    widgets::{Block, Paragraph, Scrollbar, ScrollbarOrientation, ScrollbarState},
};

use crate::{components::toast::ToastState, state::AppState};

pub fn render(
    frame: &mut Frame,
    area: Rect,
    state: &mut AppState,
    session_id: String,
    toast: &mut ToastState,
) {
    state.session.ensure_loaded(&session_id, &state.sync);

    let sidebar_visible = state.session.view.sidebar_visible(area.width);

    let [main, sidebar] = if sidebar_visible {
        Layout::horizontal([
            Constraint::Min(0),
            Constraint::Length(42),
        ])
        .areas(area)
    } else {
        [area, Rect::default()]
    };

    render_main(frame, main, state, &session_id);

    if sidebar_visible {
        crate::session::sidebar::render(frame, sidebar, state, &session_id);
    }

    toast.render(frame, main, &state.theme.current);
}

fn render_main(frame: &mut Frame, area: Rect, state: &mut AppState, session_id: &str) {
    let [feed, bottom] = Layout::vertical([
        Constraint::Min(0),
        Constraint::Length(bottom_height(state)),
    ])
    .areas(inner_x(area, 2));

    render_feed(frame, feed, state, session_id);
    render_bottom(frame, bottom, state, session_id);
}

fn render_feed(frame: &mut Frame, area: Rect, state: &mut AppState, session_id: &str) {
    let theme = &state.theme.current;
    let messages = state.sync.messages(session_id);

    let lines = crate::session::renderers::render_messages_to_lines(
        messages,
        &state.sync,
        &state.session.view,
        theme,
    );

    let paragraph = Paragraph::new(lines)
        .style(Style::default().fg(theme.text))
        .scroll((state.session.view.scroll_offset, 0));

    frame.render_widget(paragraph, area);

    if state.session.view.show_scrollbar {
        let mut scrollbar = ScrollbarState::new(state.session.feed_height as usize)
            .position(state.session.view.scroll_offset as usize);

        frame.render_stateful_widget(
            Scrollbar::new(ScrollbarOrientation::VerticalRight)
                .track_symbol(Some("│"))
                .thumb_symbol("█"),
            area,
            &mut scrollbar,
        );
    }
}

fn render_bottom(frame: &mut Frame, area: Rect, state: &mut AppState, session_id: &str) {
    let mut constraints = Vec::new();

    if state.session.daemon_run.is_some() {
        constraints.push(Constraint::Length(1));
    }
    if !state.session.view.permissions.is_empty() {
        constraints.push(Constraint::Length(5));
    } else if !state.session.view.questions.is_empty() {
        constraints.push(Constraint::Length(5));
    }
    if state.session.view.visible_prompt {
        constraints.push(Constraint::Length(4));
    }

    if constraints.is_empty() {
        return;
    }

    let chunks = Layout::vertical(constraints).split(area);
    let mut idx = 0;

    if state.session.daemon_run.is_some() {
        crate::session::daemon::render_banner(frame, chunks[idx], state);
        idx += 1;
    }

    if !state.session.view.permissions.is_empty() {
        crate::session::permissions::render(frame, chunks[idx], state);
        idx += 1;
    } else if !state.session.view.questions.is_empty() {
        crate::session::questions::render(frame, chunks[idx], state);
        idx += 1;
    }

    if state.session.view.visible_prompt {
        crate::components::prompt::render(frame, chunks[idx], state, Some(session_id));
    }
}

fn bottom_height(state: &AppState) -> u16 {
    let mut h = 0;
    if state.session.daemon_run.is_some() {
        h += 1;
    }
    if !state.session.view.permissions.is_empty() || !state.session.view.questions.is_empty() {
        h += 5;
    }
    if state.session.view.visible_prompt {
        h += 4;
    }
    h.max(1)
}

fn inner_x(area: Rect, padding: u16) -> Rect {
    Rect {
        x: area.x.saturating_add(padding),
        y: area.y,
        width: area.width.saturating_sub(padding * 2),
        height: area.height,
    }
}
```

## 18. Internal plugin and slot replacement

The current TypeScript plugin API cannot survive unchanged because it exposes:

```text
JSX.Element
SolidPlugin
CliRenderer
ParsedKey
RGBA from @opentui/core
SlotMode from @opentui/core
```

If Bun is removed 100%, then TypeScript TUI plugins also have to be removed or redesigned. The safest migration is:

1. Convert all current “internal plugins” into Rust feature modules.
2. Keep “slot” terminology for layout extensibility.
3. Expose a Rust trait for internal features.
4. For external extensibility, use data-driven config first, not runtime JS/TS UI plugins.

### `plugins/slots.rs`

```rust
use std::collections::HashMap;

use ratatui::{prelude::*};
use serde_json::Value;

use crate::state::AppState;

#[derive(Debug, Clone, Copy, PartialEq, Eq, Hash)]
pub enum SlotName {
    App,
    HomeLogo,
    HomeBottom,
    HomeFooter,
    SessionPrompt,
    SessionPromptRight,
    SidebarTitle,
    SidebarContent,
    SidebarFooter,
    ShellLeftTabs,
    ShellLeftActivePane,
    ShellCenterFeed,
    ShellFooter,
}

#[derive(Debug, Clone, Copy)]
pub enum SlotMode {
    Additive,
    Replace,
    SingleWinner,
}

pub trait TuiFeature {
    fn id(&self) -> &'static str;

    fn render_slot(
        &mut self,
        slot: SlotName,
        frame: &mut Frame,
        area: Rect,
        state: &mut AppState,
        props: &Value,
    ) -> bool {
        let _ = (slot, frame, area, state, props);
        false
    }

    fn update(&mut self, action: &crate::action::Action, state: &mut AppState) {
        let _ = (action, state);
    }
}

pub struct SlotRegistry {
    features: Vec<Box<dyn TuiFeature>>,
    modes: HashMap<SlotName, SlotMode>,
}

impl SlotRegistry {
    pub fn new() -> Self {
        Self {
            features: Vec::new(),
            modes: HashMap::new(),
        }
    }

    pub fn register(&mut self, feature: impl TuiFeature + 'static) {
        self.features.push(Box::new(feature));
    }

    pub fn render(
        &mut self,
        slot: SlotName,
        frame: &mut Frame,
        area: Rect,
        state: &mut AppState,
    ) -> bool {
        self.render_with(slot, frame, area, state, Value::Null)
    }

    pub fn render_with(
        &mut self,
        slot: SlotName,
        frame: &mut Frame,
        area: Rect,
        state: &mut AppState,
        props: Value,
    ) -> bool {
        let mode = self.modes.get(&slot).copied().unwrap_or(SlotMode::Additive);
        let mut rendered = false;

        for feature in &mut self.features {
            let did_render = feature.render_slot(slot, frame, area, state, &props);

            match mode {
                SlotMode::Additive => rendered |= did_render,
                SlotMode::Replace | SlotMode::SingleWinner if did_render => return true,
                _ => {}
            }
        }

        rendered
    }
}
```

## 24. Highest-risk areas and mitigations

### Risk 1 — Pixel-perfect OpenTUI flexbox to Ratatui manual layout

OpenTUI currently uses JSX/flexbox-like primitives. Ratatui uses explicit `Rect` layout. This is good for control, but it means every layout must be intentionally recreated.

Mitigation:

* Preserve existing breakpoints exactly.
* Write layout unit tests for rect splits.
* Snapshot buffers at 80/100/120/160/200 columns.
* Port padding, gap, border, and background rules before porting content.

### Risk 2 — TypeScript TUI plugin API cannot remain

The current `packages/plugin/src/tui.ts` exposes JSX, Solid plugins, OpenTUI renderer, OpenTUI key events, and OpenTUI color types. With Bun/OpenTUI/Solid gone, this API is incompatible.

Mitigation:

* Convert internal plugins to Rust features.
* Replace external TUI plugin rendering with data-driven extension points.
* Keep plugin config compatibility where possible:

  * `plugin`
  * `plugin_enabled`
  * keybind config
  * theme config
* Mark TypeScript TUI plugins as removed/deprecated in release notes.

### Risk 3 — Prompt editing and Unicode width

The current code uses `Bun.stringWidth`. Rust must handle wide characters and grapheme clusters correctly.

Mitigation:

* Use `unicode-width`.
* Use `unicode-segmentation`.
* Add tests for emoji, CJK, combining marks, backspace, delete, left/right movement.

### Risk 4 — Terminal restore on panic

The current app manually restores alternate screen and cursor visibility. Rust must be even stricter.

Mitigation:

* `Drop` implementation on `Tui`.
* Panic hook that restores terminal.
* `color-eyre`/custom hook.
* Integration test that simulates startup failure.

### Risk 5 — Mouse/selection/copy behavior

OpenTUI had selection/copy integration. Ratatui/Crossterm gives lower-level events.

Mitigation:

* Implement basic mouse support first.
* Preserve keyboard copy commands first.
* Add selection/copy later if truly required.
* Use clipboard crate such as `arboard` or OSC52 behind feature flags.

### Risk 6 — Bun scripts may hide non-TUI responsibilities

Many Bun scripts are build, publish, schema, smoke, and CI helpers.

Mitigation:

* Port them to `xtask`.
* Do not delete until equivalent Cargo command exists.
* Add one-to-one script mapping in CI.

## 26. Bottom-line recommendation

The clean migration path is:

```text
OpenTUI/Solid/Bun TUI
    ↓
Rust AppState + Action loop
    ↓
Ratatui render functions
    ↓
Crossterm event stream
    ↓
Cargo/xtask/CI/Nix
```

Do not preserve the current TypeScript TUI plugin runtime if the goal is truly **Bun 100% removed**. Treat the current internal plugins as product features and port them to Rust modules. Preserve the visual grammar: same logo, same gold, same header, same footer hints, same shell breakpoints, same prompt behavior, same session/sidebar behavior. The exact implementation should become simpler, more deterministic, and easier to test because Ratatui renders from explicit state into a buffer, which is ideal for snapshot tests.
